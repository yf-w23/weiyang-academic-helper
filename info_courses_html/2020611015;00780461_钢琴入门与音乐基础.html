课程内容简介：
《 钢琴入门与音乐基础 》是面对全校学生开设的数码钢琴教学课程，从零开始，通过实践，在循序渐进的学习过程中，掌握一定的技能，在进一步扩大知识面的同时，让学生经过自身的努力，能够独立挖掘自己喜欢但又从来没有面对过的层面，对音乐有一个直观地了解，是传统钢琴教学与现代高科技相结合而发展起来的一门新的键盘课程，也是为渴望学习钢琴弹奏但缺乏音乐基础知识的大学生开设的入门课程。
Course Description
进度安排：
第一讲
1. 钢琴发展史简介、认识电钢琴
2. 课程综述
3. 坐姿；手型
4. 指法
5. 认识键盘
6. 音名、唱名
7. 中央C手位
8. 音符（四分、二分、附点二分、全音符）
9. 拍号（四二、四三、四四拍）
10. 认识五线谱；地标音（中央C、高音G、低音F）
11. 音（低音F~高音G）
12. 反复记号（从头反复）
第二讲
1. 休止符（四分、二分、全休止符）
2. 延音线/同音连线
第三讲
1. 音符（八分）、休止符（八分）
2. 地标音（低音C、中央C、高音C）
3. C大调手位
4. 半音、全音；音阶；临时记号（升、降、还原记号）
5. 加线、加间（高音谱表下方、低音谱表上方）
第四讲
1. 力度记号（p、mp、mf、f）
2. 音（低音C~E、高音A~C）
3. 连线、连奏
第五讲
1. 调号（C、G、F大调）
2. 音阶（C、G、F大调5音音阶）
3. 音（低音F~C、高音C~G）
第六讲
1. 音程（旋律音程、和声音程）
2. 和弦（三和弦、七和弦）
3. 分解和弦
第七讲
1. 和弦转位
2. 固定低音和弦
3. 力度记号（渐强、减弱记号）
4. 重音记号
5. 双手交叉弹奏
7. 反复记号（段落反复、两个结尾的反复）
第八讲
1. 二音连线
2. 音阶（大、小调1个八度音阶）
3. 跳音记号、手腕断奏
第九讲
1. 地标音（Low C、High C）
2. 加线、加间（高音谱表上方、低音谱表下方）
3. 切分音/切分节奏
4. D、bB大调手位
5. 半音阶
第十讲
1. 三音连线
2. 和弦（属七和弦）
3. 音符（附点四分音符）
4. 拍号（八三、八六拍）
5. A大调手位
第十一讲
1. 踏板（延音踏板）
2. 拍号（四四、二二拍）
第十二讲
1. bE大调手位
2. 同音换指
3. 穿指
4. 音符（十六分音符）
第十三讲
1. 反复记号（D. S. al fine、D. C. al fine）
2. 颤音
3. 三连音
第十四讲
1. 琶音
2. 装饰音
3. 保持音
4. 终止和弦
第十五讲
复习考试曲目
第十六讲
期末考试
考核方式：
考试（音乐会形式）
