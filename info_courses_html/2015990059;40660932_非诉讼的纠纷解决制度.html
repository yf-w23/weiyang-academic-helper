课程名：
非诉讼的纠纷解决制度
总学时：
32

课程内容简介：
本课程将以当下中国的“国家、社会与法”为关切，采用“倒转瞭望镜头”的方法，探寻纷繁现象的历史根源。为避免教学内容散漫，将有限课时集中于若干关键词的研讨。本课程围绕“家”、“财产”、“纠纷解决”、“司法审判”、“国家”、“意识形态”与“法”等关键词，选取合适的教学材料，供选课同学课前阅读，课上讨论，课后撰写书评。
Course Description
进度安排：

  一、导论：“坐标系”理论（2课时）

 

二、家庭与宗族（6课时）

阅读材料

1.[日]寺田浩明：《清代传统法秩序》，王亚新监译，广西师范大学出版社2023年版，第18-64页（第一章“人与家”）；

2.[日]滋贺秀三：《中国家族法原理》，张建国、李力译，商务印书馆2013年版，第26-155页（第一章“基本的诸概念”）；

3.瞿同祖：《中国法律与中国社会》，商务印书馆版2010年版，第1-159页（第一章“家族”、第二章“婚姻”）；

4.[法]马克Ÿ布洛赫：《封建社会（上卷）》，张绪山译，商务印书馆2019年版，第215-248页（第九章“家族群体的连带关系”、第十章“家族关系纽带的特性和变迁”）

 

三、财产：土地“所有”观念（6课时）

阅读材料

1.[日]寺田浩明：《清代传统法秩序》，王亚新监译，广西师范大学出版社2023年版，第65-138页（第二章“生业与财产”）；

2.[日]寺田浩明：《权利与冤抑：寺田浩明中国法史论集》，王亚新等译，清华大学出版社2012年版，第72-88页（“中国近世土地所有制研究”）、215-233页（“权利与冤抑——清代听讼和民众的民事法秩序”前半部分）；

3.[日]岸本美绪：《风俗与历史观：明清时代的中国与世界》，梁敏玲等译，广西师范大学出版社2022年版，第209-232页（第九章“买卖土地与买卖人口”）；

4.[法]马克Ÿ布洛赫：《封建社会（上卷）》，张绪山译，商务印书馆2019年版，第274-292页（第十二章“采邑”）、第313-341页（第十四章“采邑变成附庸的世袭财产”）、第387-404页（第十八章“庄园”）；

四、纠纷解决与秩序建构（6课时）

阅读材料

1.[日]寺田浩明：《清代传统法秩序》，王亚新监译，广西师范大学出版社2023年版，第178-306页（第四章“秩序、纷争和诉讼”、第五章“听讼——审判与判决的社会基础”）；

2.[日]寺田浩明：《权利与冤抑：寺田浩明中国法史论集》，王亚新等译，清华大学出版社2012年版，第233-274页（“权利与冤抑——清代听讼和民众的民事法秩序”后半部分）；

3.[日]滋贺秀三：《清代中国的法与审判》，熊远报译，江苏人民出版社2023年版，第243-283页（第四章“民事审判中法源的概括性研究”）；

4. 李秀清：《日耳曼法研究（修订版）》，社会科学文献出版社2018年版，第364-428页（第九章“解决纠纷的组织及机制”）

 

五、司法审判：正义的多种面孔（4课时）

阅读材料

1.[日]寺田浩明：《清代传统法秩序》，王亚新监译，广西师范大学出版社2023年版，第307-402页（第六章“断罪——犯罪的处罚与判决的统一”）；

2.[日]滋贺秀三：《清代中国的法与审判》，熊远报译，江苏人民出版社2023年版，第144-242页（第三章“判决确定力观念的不存在”）；

六、国家与意识形态（4课时）

阅读材料

1.赵鼎新：《儒法国家：中国历史新论》，第55-86页（第二章“西周的政治体制及其衰落”）、第298-336页（第九章“西汉与儒法国家的来临”）、第377-395页（第十二章“新儒学与‘儒家社会’的到来”）； 

2.苏力：《大国宪制：历史中国的制度构成》，北京大学出版社2018年版，第442-489页（第十章“作为制度的皇帝”）； 

3.金观涛、刘青峰：《开放中的变迁：再论中国社会超稳定结构》，法律出版社2011年版，第181-235页（第五章“意识形态更替机制”）；

4.[法]菲利普Ÿ内莫：《古典与中世纪政治思想史》，张竝译，华东师范大学出版社2021年版，第905-961页（第六章“中世纪末期：近代国家概念的初生”）

 

七、总结：反思“法”概念（4课时）

阅读材料

   [日]寺田浩明：《清代传统法秩序》，王亚新监译，广西师范大学出版社2023年版，第403-495页（第七章“法律、权力和社会”、第八章“传统中国法与近代法”、终章“跨越文明的法论述方式”）
