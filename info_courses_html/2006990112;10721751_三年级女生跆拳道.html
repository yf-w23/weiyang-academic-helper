课程内容简介：
一、介绍该项目的起源和发展。 二、学习理论及裁判规则，主要练习基本技术和基本战术，反复地练习达到较熟练地掌握技能和规则，并能参加比赛。通过学习和练习，促进学生的身体，心理和全面人格的发展。使同学们达到防身、健身的目的。 三、通过跆拳道的训练，可以培养练习者坚韧不拔，勇敢无畏，顽强坚毅的意志品质，练习者从开始就养成谦虚、宽容、礼让的高尚品德和尊师重道、讲礼守信、见义勇为的情操。
Course Description
进度安排：

1、基本理论知识讲解；

2、基本步法；

3、基本进攻技术、反击技术、连击技术；

4、进攻、防守连击技术；

4、基础战术运用。

 
考核方式：
1、基本技术； 2、连击踢靶技术。
