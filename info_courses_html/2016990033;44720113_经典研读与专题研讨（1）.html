课程内容简介：
2008年清华大学抢救收藏了一批战国竹简，学界通称“清华简”。清华简总数约2500枚，主要以战国时期楚文字书写，内容以古书为主，涉及经、史、子、天文数术等中国传统文化的核心内容，价值重大。其中少量篇目可与传世文献相对照，大量内容是前所未见的先秦佚籍。清华简研究也是近十年来出土文献与古文字研究的学术最前沿。 本课程以精读清华简文本为基础，采取由字、词、句到章、篇、卷的循序研读法，通过对文字的识读、文句的疏解以及相关史事与传世文献的系联考证，帮助同学逐步掌握自主阅读简帛古书和传世文献的基本方法，提升阅读古典文献的能力，并开展探研学术问题的初步实践。2026年春季学期课程将以清华简《系年》为精读对象。
Course Description
进度安排：

 

一、清华简《系年》共有简138支，凡3875字（含合文与重文），以战国楚文字书写。内容分为二十三章，所载史事上起周武王克商，下至战国早期的三晋与楚的大战等，时间跨度达600余年。通读本篇有助于熟悉掌握楚文字常用字，并搭建两周历史的基本框架。精读之前，需要了解其整体体例和基本性质。

二、课程形式以报告讨论为主。根据选课人数将《系年》二十三章内容进行分组，同学以组为单位合作研讨报告，提出疑难字、词、句及相关史事，重点进行讨论。

三、课程要求一定量的课外阅读，尤其是配合简文，阅读内容相关的《左传》《国语》《史记》等文献资料。
