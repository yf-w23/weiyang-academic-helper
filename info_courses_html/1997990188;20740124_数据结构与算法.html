Course Description
Data structures and algorithms play a central role in computer science and are the cornerstones of efficient solutions using computers in various applications. This course explores the principles and methods in the design and implementation of various data structures. Students will also learn the essential skills for algorithm design and performance analysis. Topics covered by this course range from fundamental data structures and algorithms to recent research results.
进度安排：
本课程于每年春季学期，由邓俊辉老师讲授；首先面向未央书院的同学，同时欢迎所有书院的同学；相对于面向计算机系同学开设的同名系列课，难度稍低。按照自编讲义，分为概述、线性结构、半线性结构、非线性结构等模块，其中涉及图结构的部分不做要求。平时分批次地设有编程作业，统一在实验平台（https://dsa.cs.tsinghua.edu.cn/oj/）上完成。期末设有卷面考试，与平时成绩一并汇总后得出总评成绩。
