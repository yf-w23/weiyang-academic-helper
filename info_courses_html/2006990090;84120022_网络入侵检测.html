课程简介：
随着互联网的迅速发展，网络中的攻击行为也越演越烈，其强度和破坏程度也随之增强。作为一种检测网络中攻击行为的重要手段，网络入侵检测技术主要通过收集网络中（包括主机、中间设备如路由器等）的相关信息并对其进行分析，从而发现网络中的入侵或攻击行为。本课程围绕网络入侵检测技术进行讲授，主要涉及以下相关内容：一. 入侵检测技术概述：1.课程概述和相关要求；2. 网络攻击/入侵；3. 入侵检测技术概述；二. 入侵检测核心技术：4. 入侵检测的数据源；5. 网络入侵检测方法；6. 开源入侵检测系统介绍；7. 入侵检测系统的标准与测评；三. 新技术专题研讨：8. 基于机器学习的网络入侵/异常检测研究；9. 基于大数据技术及平台的网络入侵/异常检测研究；10. 基于软件定义网络（SDN）的网络入侵/攻击检测研究。 本课程除讲授入侵检测的基础知识之外，还将结合本领域的相关顶级会议和期刊论文，介绍本领域的最新进展。本课程采用多种教学方式，如课堂论文研讨、课后文献阅读作业、课程项目等加深学生对该领域的了解和认识，并提高研究生必备的科研技能。
英文课程简介：
With rapid development of the Internet, network attacks are becoming more popular, and their intensity and degree of the damage are increasing. As one of the important techniques to detect network attacks, network intrusion detection collects and analyzes the related information in the network (including hosts and network devices, e.g. routers) in order to identify the intrusion and attack behaviors in the network. This course focuses on the network intrusion detection technique, including: 1. Brief introduction of intrusion detection: (1) Course introduction and related requirements; (2) Network attacks/intrusion; (3) Brief introduction of intrusion detection; 2. Core techniques of intrusion detection: (4) Data source of intrusion detection; (5) Network intrusion detection Methods; (6) Open source intrusion detection system; (7) Standardization and evaluation of intrusion detection system; 3. Discussion of new research directions: (8) Machine Learning based network intrusion/anomaly detection; (9) Big data based network intrusion/anomaly detection; (10) Software Defined Networking (SDN) based Network intrusion/attack detection; Besides the basic knowledge of network intrusion detection, this course will introduce the latest progress of this field by investigating the related papers of the top conferences and journals in this field. I this course, we will use several teaching methods, such as paper seminars, homework of paper reading and course project, in order to deepen students' knowledge and understanding of this field and improve the necessary research skills of graduate students.
