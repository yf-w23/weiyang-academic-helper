课程内容简介：
本课程为思想政治理论课，将系统讲授毛泽东思想和中国特色社会主义理论体系。讲授方式为大班授课。 课程将带领大家系统学习毛泽东思想、邓小平理论、“三个代表”重要思想、科学发展观等重要思想理论成果，同大家一起走进中国化马克思主义的思想世界，感悟百年来中国共产党人的思想历程，感悟中国化马克思主义何以能行的内在逻辑。
Course Description
进度安排：
  • 本课程将主要依据《毛泽东思想和中国特色社会主义理论体系概论》教材要求完成教学进度安排，系统讲述毛泽东思想和中国特色社会主义理论体系的相关知识点。具体课程进度如下，可能视情况有所调整。
  • 第1讲《马克思主义中国化的思想进程与理论成果》
  • 第2讲《青年毛泽东的“人生选择”》
  • 第3讲《毛泽东与中国革命的“星星之火”》
  • 第4讲《毛泽东与中国革命道路的开辟》
  • 第5讲《新民主主义革命理论（上）》
  • 第6讲《新民主主义革命理论（下）》
  • 第7讲《社会主义改造理论》
  • 第8讲《毛泽东与新中国社会主义建设（上）》
  • 第9讲《毛泽东与新中国社会主义建设（下）》
  • 第10讲《毛泽东的“历史遗产”与毛泽东思想的历史地位》
  • 第11讲《历史转折中的“突围”与“破局”》
  • 第12讲《邓小平理论的创新与开拓》
  • 第13讲《“三个代表”重要思想与迈向21世纪的中国特色社会主义》
  • 第14讲《科学发展观的科学内涵与主要内容》
  • 第15讲《中国共产党光荣与梦想的远征》
