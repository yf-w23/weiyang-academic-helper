视听语言与短片创作

课程内容简介：
一直以来，“视听语言”是影视传媒院校的专业必修课，是广播电视编导及电视摄影专业开设的专业基础课。这门 课从影像、声音、剪辑三个角度着手，注重对影像创作中的画面和声音元素的使用技巧以及创作理念的研究。视 听语言为本课程奠定了知识基础，剧情短片创作作为综合性体现学生视听语言掌握程度的方式，是本课程主要的 实践和考察部分。在短片创作部分，本课程将继续采用国际先进的影视创作教学理念，以实践教学为主，对学生 的剧情短片创作学习和演练进行指导，使学生对剧情片的创作理念和技巧以及相关的导演、编剧、摄制以及制片 等工作流程有比较全面的领会与掌握，并培养团队合作意识。 通过收集来自不同学科、不同年级的学生反馈，教师在教学中不断总结经验解决学生在实践中所遇到的问题。在 不断调整推进教学上的升级优化，积累跨学科教学经验的同时，我们深刻地意识到，对于普通高等院校的跨学科 学生来说，现有的《视听语言》教材专业性过高，内容设计偏重于理论讲授，对于没有相关知识储备的跨学科学 生来说，教材理论过于艰深晦涩，在教学中的指导意义及可操作性不强，不能完全适用于跨学科学生的学习。因 此在本次课程中，我们通过大量影视案例来引导学生学习视听语言的知识，利用短片创作部分让学生可以通过实 践来检验对视听语言知识的掌握情况。 通过这一门课程，可以培养学生掌握现代影像拍摄理念、方法与技巧，具备熟练地运用视听语言完成思想和内容 表达的专业素质，摆脱固有的文字传播的影响，在视频化生存状态的当下，掌握欣赏视听语言的能力，并进一步 拥有运用视听语言技能进行视频拍摄、剪辑的制作技能。
Course Description
进度安排：

第一章 电影的镜头语言（上）（6学时）

重点内容：

1.知识讲授（3学时）

2.经典片例解析（3学时）

 

第二章 电影的镜头语言（下）（6学时）

重点内容：

1.知识讲授（3学时）

2.经典片例解析（3学时）

 

第三章 长镜头叙事与场面调度 （9学时）

重点内容：

1. 知识讲授、经典片例解析（2学时）

2. 构思、讨论作业1（2学时）

3. 拍摄、创作作业1（3学时）

4. 拉片、讲评作业1（2学时）

作业1：拍摄90秒“长镜头”剧情短片，要求：

（1）长度90秒；

（2）体现导演明确的场面调度；

（3）在单一长镜头内体现叙事视角的关系与变化；

（4）在镜头内建立人与人及人与环境的关系，并通过关系的呈现和变化讲述一个相对独立的故事。

 

第四章 推动叙事的视觉化细节元素（9学时）

重点内容：

1. 知识讲授、经典片例解析（2学时）

2. 构思、讨论作业2（2学时）

3. 拍摄、创作作业2（3学时）

4. 拉片、讲评作业2（2学时）

作业2：拍摄3分钟“细节”剧情短片，要求：

（1）长度3分钟 ；

（2）设置视觉与听觉的细节；

（3）围绕该细节讲述相对独立的故事；

（4）力求细节带动人物出场，表现人物关系。

 

第五章  推动故事并吸引观众的悬念（9学时）

1. 知识讲授、经典片例解析（2学时）

2. 构思、讨论作业3（2学时）

3. 拍摄、创作作业3（3学时）

4. 拉片、讲评作业3（2学时）

作业3：拍摄3分钟“悬念”剧情短片，要求：

（1）长度3分钟；

（2）以细节设计悬念；

（3）特写与全景两极镜头结合运用，利用光影及音响营造气氛。

第六章  具有叙事意义的结构语言（9学时）

1. 知识讲授、经典片例解析（2学时）

2. 构思、讨论作业4（2学时）

3. 拍摄、创作作业4（3学时）

4. 拉片、讲评作业4（2学时）

作业4：拍摄4分钟“结构”剧情短片，要求：

（1）长度4分钟；

（2）采用独特的结构方式讲述故事；

（3）赋予故事结构以叙事意义；

（4）以结构为统领，以体现人物关系为重要目的，综合运用细节、悬念及音响等多方面电影元素。

 

 
考核方式：
考核办法：短片创作 出勤率+课堂报告+短片作品=100%。
