课程内容简介：
通过对计算和智能计算的基本原理的学习，在课程中学习Python编程，实践机器智能的实验，完成机器智能的课程项目。 通过对机器智能系统的实践，在实践中掌握TensorFlow2机器学习技术，实践TensorFlow2的人工智能应用，如计算机视觉、语音识别等。 课程前沿模块探索基于大语言模型LLM的机器智能，包括：现代深度学习架构，transformer，LLM前沿研究进展，如高效率推断等。课程选题具有一定挑战性与前沿研究性。
Course Description
This course is teaching students to use Python, and TensorFlow2 in learning the principle of machine intelligent system. Python is a popular language in data science and machine learning and TensorFlow2 is a popular library for machine learning using Python as frontend. This course introduces Python in programming practice, and students will be instructed to programming practice based on Jupyter notebook. Tensorflow2 is a machine learning library. Students will be instructed to use TensorFlow2 and Keras to build deep models to solve speech recognition, computer vision task and natural language processing etc. The course explores machine intelligence based on the Large Language Model (LLM), including: modern deep learning architectures, Transformers, and cutting-edge research advances in LLM, such as high-efficiency inference. The topics selected for the course are challenging and cutting-edge research.
