课程内容简介：
本课程针对天线的基础知识进行讲授，理论结合实际，通过身边的天线实例，学习天线对电磁波的发射与接收； 理论部分，面向有学术研究需求的同学，通过一门课可以掌握天线的基础理论； 工程部分：面向有工业界发展的同学需求，学会设计简单的天线，应用于实际无线通信系统等。
Course Description
进度安排：

1、天线发展史、天线科普

2、天线理论基础：电磁波的辐射

3、天线参数、如何评价一款天线

4、天线链路计算：天线在无线系统中的地位

5、不同类型天线工作原理：偶极天线、环天线、孔径天线、微带天线、反射面天线

6、天线阵列

7、天线应用：手机天线、基站天线、雷达天线、广播天线等

8、天线前沿知识
