课程简介：
以超级计算机为代表的高性能计算技术已成为科学计算等前沿学科的重要计算方法。本课程重视实际问题与并行计算理论知识结合，基于最前沿领域扩展知识面。成绩主要由研讨（30%）、上机实践（30%）和研究报告（40%）组成，不安排考试。课程主要包括三方面内容： 1、讲授前沿计算科学与工程领域发展趋势和对并行计算的需求，涉及地球科学、生命科学、材料科学等重要领域；此外，还会涉及人工智能、量子计算等前沿领域与高性能计算融合的最新发展趋势。 2、讲授高性能计算发展趋势、最新软硬件技术和优化方法，涉及高性能计算处理器（如CPU、GPU、国产处理器）、国内外领先超级计算机（神威、天河、Summit、Fugaku）等； 3、讲授高性能计算系统的并行编程方法，通过文献阅读、上机实践、撰写研究报告等，形成并行计算思维，提高动手实践能力。
英文课程简介：
High performance computing (HPC) has become an important solution for various applications. This course mainly focuses on resolving key challenges from these applications by using parallel computing methods. 1. State-of-the-art HPC solutions in various applications including earth science, life science, material science, deep learning, quantum computing, etc. 2. Sophisticated HPC techniques on both software and hardware levels, including processors (CPU, GPU, etc.) and supercomputing systems (Sunway, Tianhe, Summit, Fugaku, etc.). 3. Parallel programming methods based on HPC systems. HPC Programing environments are provided for practice.
