课程简介：
课程：《基于Python的机器学习实践》 1. 课程目标与定位 本课程旨在为学生提供扎实的机器学习理论基础，并重点培养基于Python语言的实际编程与应用能力。学生将通过大量的实践项目，深入理解从数据处理到模型部署的完整机器学习流程，掌握解决真实世界问题的能力。 2. 面向对象 本课程面向具备基础编程能力与数学素养的高年级本科生和研究生。对数据科学、人工智能或相关领域有浓厚兴趣的学生，希望通过实践掌握机器学习核心技能的学生。 3. 课程内容概览 本课程内容涵盖统计机器学习和深度学习两大核心领域，理论与实践并重，并充分利用Python的强大生态系统。 1) 基础与经典机器学习 - 机器学习概览：机器学习的图景、端到端项目流程、数据处理、特征工程与选择。 - 经典算法：深入学习分类、回归、聚类、降维等任务中的核心算法，包括： * 线性模型 (如：线性回归、逻辑回归) * 支持向量机 (SVM) * 决策树与集成学习 (如：Random Forests) * 无监督学习技术 (如：K-Means, PCA) 2) 深度学习与前沿应用 - 神经网络基础：全连接神经网络 (FCNN) 的构建、训练深度模型、TensorFlow, PyTorch与Keras实践。 - 计算机视觉 (CV): 掌握卷积神经网络 (CNN) 的原理与应用，如图像分类、目标检测、图像分割。 - 自然语言处理 (NLP)：学习使用循环神经网络 (RNN) 和Attention机制，掌握文本分类、语言模型和文本生成等技术。 - 前沿模型：介绍Transformer模型，Diffusion Models等。 4. 实践工具与项目 课程将以Hands-on的方式进行，重点使用以下Python库： - 数据处理与科学计算： NumPy, Pandas - 经典机器学习： Scikit-learn - 深度学习框架： Keras (基于TensorFlow和PyTorch) 学生将通过多个真实世界的案例研究进行实践，例如：分析共享单车趋势、电影评论情感分析、客户群体细分、预测股票价格、以及大规模模型的训练与部署。
英文课程简介：
Course: Practical Machine Learning with Python 1. Objectives: Equip students with solid ML theory and hands-on Python skills. Cover the full ML pipeline from data preparation to model deployment, enabling solutions to real-world problems. 2. Target Audience: Upper-division undergraduates or graduate students with basic programming and math skills. Ideal for those interested in data science, AI, or related fields seeking practical ML mastery. 3. Content Overview: Covers Statistical ML and Deep Learning, emphasizing theory and Python implementation. 1） Fundamentals & Classical ML - ML Landscape: Workflow, data processing, feature engineering, selection. - Core Algorithms: Classification, Regression, Clustering, Dimensionality Reduction, including SVM, Decision Trees, Random Forests, Linear Models, K-Means, PCA. 2） Deep Learning & Advanced Applications - Neural Networks: FCNN construction, training, TensorFlow/Keras applications. - Computer Vision: CNNs for image classification, object detection, segmentation. - NLP: RNNs, Attention, text classification, language modeling, text generation. - Advanced Models: Transformer, Diffusion Models. 3) Practical Tools & Projects - Python Libraries: NumPy, Pandas, Scikit-learn, Keras/TensorFlow/PyTorch. - Hands-on Cases: Bike-sharing trends, movie review sentiment, customer segmentation, stock forecasting, large-scale model training and deployment.

进度安排：

第 1 周：课程概览与 Python 机器学习生态 Python 科学计算与机器学习生态体系概述；NumPy 与 Pandas 基础 第 2 周：统计机器学习基础 数据预处理与探索性分析、特征工程、超参数调优、模型选择与评估；scikit-learn 框架 第 3 周：统计机器学习方法（一）：分类 常见分类模型及其原理与实践（基于 scikit-learn） 第 4 周：统计机器学习方法（二）：回归 回归模型原理、实现与应用（基于 scikit-learn） 第 5 周：统计机器学习方法（三）：聚类 聚类分析方法与典型应用（基于 scikit-learn） 第 6 周：统计机器学习方法（四）：降维与集成学习 降维技术、集成学习方法及过拟合与泛化问题（基于 scikit-learn） 第 7 周：统计机器学习综合案例分析 基于 scikit-learn 的完整建模流程与综合实践 第 8 周：深度学习基础与主流框架 深度学习基本概念；TensorFlow、PyTorch 与 Keras 框架概述 第 9 周：深度神经网络训练与优化 模型训练流程、损失函数、优化算法与调参策略（基于 Keras） 第 10 周：卷积神经网络与图像任务 CNN 结构原理与图像分类实例（基于 Keras） 第 11 周：迁移学习与预训练模型 迁移学习方法、常用预训练模型及应用实践（基于 Keras） 第 12 周：循环神经网络与序列建模 RNN 及其变体在序列数据建模中的应用（基于 Keras） 第 13 周：自然语言处理基础 文本表示方法与文本分类任务（基于 Keras） 第 14 周：Attention 机制与 Transformer 模型 Transformer 架构原理及典型应用示例（基于 Keras） 第 15 周：生成模型与前沿应用 大型语言模型（GPT）与扩散模型（Diffusion）的基本原理与应用 第 16 周：课程总结与项目展示 课程回顾、综合项目汇报与评估
考核方式：
本课程采用过程性考核与综合性大作业相结合的方式。 平时作业：包含5-6次实验练习和编程作业。 期末大作业：学生自选或指定主题，完成一个端到端的机器学习项目。
