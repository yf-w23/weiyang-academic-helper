课程内容简介：
本课程以建立学生完整基础有机化学知识框架和有机化学逻辑思维方式为教学目的，以有机化合物结构-活性-反应性为主线，从化学键基本原理出发，认识和理解有机化合物的结构与性质、反应规律及实现有机化合物转化的途径。通过本课程的学习，学生应掌握有机化学的语言体系及有机化学的逻辑思维方式，并用于分析、解决和理解有机化学问题。 备注：为了更好的建立有机化学的理论知识框架，以及对有机化学学习的延续性，本人将进行有机化学A1、有机化学A2的贯通式教学，即连续两个学期分别开展A1和A2的教学，在A1的教学中会对A2所需的基础知识进行强化，建议选择本A1课程的同学可以在接下来的学期继续选修A2.
Course Description
进度安排：

       按照教学大纲，课程主要知识点如下，课程学习将根据知识点的难易程度通过自学和讲解结合进行，具体教学安排将根据反应类型及反应规律的主线对知识点进行优化处理，不严格按照教材的固有顺序进行教学安排。

 

一、绪论

1.  有机化学历史与发展

2.  有机化合物的分类与特征(结构和性质)：

（1）同分异构现象

（2）有机化合物的成键模式及化学键分类（共价键，离子键）；

3.   有机结构理论：

（1）原子结构，共价键

（2）Lewis结构理论、价键理论、杂化轨道理论

（3）分子轨道理论

（4）共振论、诱导效应、共轭效应

（5）键参数：键能、键长、键角、键的极性

（6）键的极性和分子极性的关系

（7）分子的偶极矩

（8）极性与化学性质的关联

（9）分子间作用力

（10）有机物的结构表示法

4.  有机反应理论：

（1）共价键的断裂方式及有机反应基本类型

（2）有机反应的基本方式：加成、消除、取代、氧化、还原等

（3）有机酸碱理论（Lewis和Bronsted酸碱理论）

（4）反应势能图

（5）反应热力学

（6）反应动力学，过渡态理论，Hammond假说

 

二、烷烃和环烷烃

1.有机化合物的命名：

（1）系统命名法（IUPAC）

（2）习惯命名法（普通命名法）

（3）衍生物命名法

2.烷烃的结构：

（1）烷烃的异构现象，构造异构

（2）链烷烃的构象，Newman投影式，构象稳定性

（3）乙烷的构象

（4）丁烷的构象

（5）高级烷烃的构象

3. 烷烃的物理性质（沸点，密度，溶解度）

4. 烷烃的化学性质：氧化反应，热解反应，异构化反应

5. 烷烃的自由基取代反应

（1）自由基链式反应机理

（2）化学键均裂能（BDE）与取代反应的热力学

（3）反应的活性分析

（4）反应的选择性分析（Hammond假说）

6. 环烷烃：

（1）分类

（2）单环烷烃的异构现象

（3）环烷烃的命名

7. 环烷烃的结构：

（1）Baeye张力学说，环烷烃燃烧热

（2）环己烷的构象

（3）十氢合萘的构象

8. 环烷烃的物理性质

9. 环烷烃的化学性质：

（1）取代反应（自由基取代反应）

（2）与卤素和卤代烃的开环反应（小环性质的特殊性）

（3）催化加氢及氧化反应

 

三、 不饱和烃

烯烃：

1.  烯烃的结构与命名：

碳的sp2杂化和烯烃的平面结构，σ键和π键

2.  烯烃的异构现象：

（1）构造异构

（2）立体异构（顺反异构），Z/E构型表示法

3.  烯烃的命名

4.  烯烃的物理性质：烯烃的熔、沸点与顺反结构的关联

5.  烯烃的化学反应：

（1）烯烃的加成反应总论：亲电/亲核/自由基机理，区域/立体选择性

（2）烯烃的亲电加成反应及其机理：加卤素、加氢卤酸、加次卤酸、加硫酸、加水、硼氢化—氧化反应、羟汞化反应，反应的区域及立体选择性（马氏规则），碳正离子重排

（3）烯烃的自由基加成反应（反马氏加成）

（4）催化加氢（立体化学，氢化热）

（5）烯烃的氧化反应：臭氧化、高锰酸钾、四氧化锇、环氧化反应

（6）与卡宾加成：环丙烷化反应

（7）烯烃a-氢的反应：烯丙基型取代反应及烯丙基自由基

（8）聚合反应（简介）

6.  烯烃的制备

 

炔烃：

1.   炔烃的结构和命名

2.   炔烃的化学反应：

（1）亲电加成反应：加卤素，加卤化氢，加水

（2）还原反应：Lindlar催化剂催化加氢，碱金属—液氨还原

（3）亲核加成：与醇，羧酸及氢氰酸反应

（4）炔烃的氧化反应

（5）炔烃的聚合反应（简介）

（6）末端炔烃的酸性及相关反应：取代和加成

3.    炔烃的制备：

（1）二卤代烷脱卤化氢

（2）末端炔烃的烷基化

 

共轭二烯：

1.   二烯烃的分类与命名

2.   共轭双烯的结构：离域体系的共振论表示法

3.   共轭二烯烃的化学反应：

（1）亲电加成：1,2-加成与1,4-加成、加成反应中的动力学/热力学控制

（2）Diels-Alder反应：分子轨道理论分析、区域与立体选择性

 

四、芳香烃

1.  芳香烃的分类、异构及命名

2.  苯的结构：

（1）苯的价键结构

（2）共振论解释（共振结构式）

（3）苯结构的分子轨道理论描述

（3）反应可逆性（磺化）、碳正离子重排（傅-克）

（1）芳环取代基对活性和区域选择性的影响

（1）稠环芳烃（萘，蒽，菲）的结构与位置编号

（2）萘：结构、亲电取代反应（活性、选择性）、氧化和还原反应

（3）蒽和菲（简介）

8.    非苯芳烃及芳香性（休克尔规则）；

七、有机波谱分析

1.         绪论：

（1）有机化合物结构解析的一般方法（近、现代）

（2）有机波谱分析概述：电磁波与分子的相互作用、分子光谱的产生

2.         红外光谱：

（1）理解红外光谱基本原理：振动转动能级、化学键的简谐振动模型

（2）各类官能团的红外吸收范围

（3）影响峰位置变化的因素：子类官能团的吸收波数范围

（4）利用红外光谱判定有机分子结构

3.    核磁共振波谱：

（1）基本原理

（2）1H NMR化学位移及其影响因素

（3）1H NMR峰型及自旋—自旋偶合

（4）1H NMR的积分

（5）13C NMR化学位移

（6）二维核磁共振简介

（7）利用NMR谱推测有机分子的结构

4.    紫外光谱：

（1）紫外光谱的基本原理：电子跃迁及其类型

（2）最大吸收波长的影响因素：Woodward规则

5.    质谱：

（1）基本原理：离子化方法、质量检测原理

（2）低分辨质谱的解析：分子离子峰（同位素峰）、EI质谱中分子离子的几种重要断裂方式

（3）高分辨质谱的解析：从精确相对分子质量推断分子式

6.    多谱图联合解析化合物结构

（2）分类和命名

（3）物理性质和波谱特征

2.       醛酮的反应模式概述：氧化还原、亲核加成、α-H的反应

3.       醛和酮的氧化还原反应：

（1）还原成醇：与负氢亲核试剂的反应、催化加氢、Meerwein-Ponndorf还原

（2）还原成亚甲基：Wolff-Kishner-黄鸣龙还原、Clemmenson还原、缩硫酮的Raney Ni还原

（4）歧化反应：Cannizarro反应

（5）氧化成酸：银镜反应

（6）氧化成酯：Baeyer-Villiger反应

4.       羰基的亲核加成反应：

（1）亲核加成反应机理与一般规律

5.       醛酮α-H的反应

（1）a-氢原子的酸性；酮式-烯醇式互变异构

（3）烷基化反应

（5）Claisen-Schmidt反应

（6）Perkin反应

（7）安息香缩合反应：极性翻转催化

（1）亲核加成反应：1,2-加成反应、1,4-加成反应（Michael加成）

（2）亲电加成反应

（3）Robinson增环反应

7.    醛和酮的合成
