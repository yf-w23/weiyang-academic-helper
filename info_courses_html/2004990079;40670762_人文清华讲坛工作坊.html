课程内容简介：
由教师和业界专家带领学生一起进行人文清华讲坛及相关内容的制作与推广。课程内容包括组织人文清 华讲坛大型活动、制作相关音视频节目和文字内容，并通过报刊、网站、社交媒体以及音视频、图文直播 等方式进行全媒体、全品类传播。 学时包括课外实践学时，上课时间36学时。
Course Description
进度安排：

 伴随人文清华讲坛的进度，进行前期培训、实习、实操。

具体包括：1纸媒图文报道的写作与推送，2微信公号内容制作与运营，3播客内容制作与运营、4客户端专区内容制作与运营、5直播实操、6视频节目制作与推广。

涉及的题材包括但不限于对教师、校友、学生进行访谈，拍摄校园风物、校园里的国宝，有趣的课程和讲座，校园体育活动、文化活动、社团活动、展览演出等等。
