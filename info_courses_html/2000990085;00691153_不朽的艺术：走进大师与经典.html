课程内容简介：
本课本门课程的主旨是在艺术史、美学理论、思想史和文化史交叉融合的基础上，贯通传统与现代，打通中国和西方，开拓和深化修课学生对人类艺术的历史变迁和文化差异的感知和认识。本课程期望，不仅给同学们以中西造型艺术经典赏析提供引导，而且开启由艺术而精神的人类心智巡礼。
Course Description
进度安排：

 

1. 线上课程：共15讲，每周更新。

2. 线下课程：本课程将有6次线下面授环节，分别为1次开课大班讲座和5次小班讨论。

3. 小班讨论时间为第 4、6、9、12、15 周的周四下午第一大节课（1:30-3:05），小班分班信息、教室地点将通过网络学堂发布。

 

 

 
考核方式：
本课程以线上讨论、线下导修和期末现场作文考试三个环节考核情况为成绩评定依据。期末现场作文考试为限制性开卷：允许使用纸质参考资料，禁止携带笔记本电脑、手机等电子信息设备进入考场。
