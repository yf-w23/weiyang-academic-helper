课程简介：
本课程讲授：（1）算法分析的基本方法；（2）常用的数据结构；（3）经典的算法设计技巧。通过本课程的学习，学生将熟练掌握渐进分析的方法、能对算法复杂度进行评价;掌握多种常用的数据结构，如栈、队列、数组、列表、哈希表、网络的使用;掌握常见的算法设计思路，如动态规划、贪心算法、分治算法等;通过上机实验，掌握调试程序的常用方法。
英文课程简介：
This course covers the following three topics: (1) Methods for algorithm analysis; (2) Common data structures; and (3) Typical algorithm design techniques.
