课程内容简介：
本课程培养建筑学专业学生在设计中的生态和环境意识，并初步掌握以解决问题为目标的整体设计方法。首先简明而系统地介绍生态学和建筑学概念和知识，然后将相关的各种设计整理成体系介绍给学生，最后从理论和实践两个方面诠释，将国内外的相关理论实践精华传授给学生。
Course Description
进度安排：
第一讲为基础概论，简单介绍理论发展和背景。 第二至7讲，分类详细介绍各种生态节能原理和技术，以及优秀案例分析。 第八讲，介绍中英联合生态建筑示范项目：张家港生态农宅。结合乡土建筑现代化和现代建筑地方化进行探讨。
