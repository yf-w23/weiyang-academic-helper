课程内容简介：
计算机领域几乎所有的系统服务，要么已经以云计算、云存储、云下载、云安全的形态提供给用户，要么正在向云端迁移。 在整个信息产业云服务化的关键时期，云服务性能优化对产业的用户体验、架构设计、规模扩张、良性发展起到核心而长期的支撑作用。 本课程讲解基于工业云计算主流系统架构的云服务性能理论、方法、工具和典型案例，主要内容包括：虚拟化与模拟器技术、 云服务性能指标框架与性能分析工具链、Linux操作系统和网络测量、云计算与云存储服务性能以及来自工业界的典型案例。 通过学习本课程，学生能够了解到云计算工业界普遍承认的性能指标、洞悉云计算系统运作的方式、学习到分析和提高云计算系统服务性能的科学方法和现实工具， 这些性能方法和工具同样适用于大型企业云计算的复杂环境，和目前工业界的云计算服务需求能够无缝对接。
Course Description
Almost all the system services in the computer area have been provided to users in the form of cloud computing, cloud storage, cloud download and cloud security, or are migrating to the cloud. In this key period of the entire information industry, cloud service performance optimization plays a core and long-term supporting role in user experience, architecture design, scale expansion and benign development of the industry. This course explains the performance theory, methods, tools, and typical cases of cloud services based on mainstream cloud computing system architectures in industrial. The main contents of this course include: brief introduction to the Linux operating system, virtualization and emulator technology, performance analysis tool chain, the metric framework of cloud service performance, the properties of processor and memory, disk and file system performance, network communication performance, the study of cloud storage’s network efficiency, and four typical cases from industry. By learning this course, students can learn about the performance metrics generally accepted by the cloud computing industry, get the insights into the internal working of the cloud computing system, learn to analyze and improve the scientific methods and practical tools for the service performance of cloud computing systems. These performance methods and tools are also applicable to complex environments of large-scale enterprise-level cloud computing, and can seamlessly meet the current industrial demands for cloud computing services.
