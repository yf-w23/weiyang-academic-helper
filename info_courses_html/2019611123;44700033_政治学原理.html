课程内容简介：
通过教师课堂讲授、阅读文献及讨论，熟悉现代政治学的核心概念、理论脉络及基本思维方式，并在中西政治传统对比的基础上，理解作为舶来品的西方政治学知识内在的认知张力，为进一步学习其它专业课程打下基础。
Course Description
进度安排：

 

本课程除第一讲“学科导引”概述政治、政治学的一般常识和基本脉络外，大体按照四个部分组织讲次内容：

 

第一，政治学最主要的三对核心概念及与之相关的争议。首先是国家与民族，其次是权力与权威，最后是理性与理念。之所以拎出来这三对概念，因为它们是我们进行政治分析的关键起点。同时，从这些基本概念开始，古今中西政治语言及政治思维就出现了一些微妙的分歧，明乎此有助于我们对西方学理与中国经验的反思。

第二，中西政治传统之间三对关键分歧，即政体与政道、民主与民本、法治与礼治。通过对这些政治传统分歧的理解，我们可以大致勾勒中华政治文明与西方政治文明之间不同的历史路径，以及因此对当代政治产生的延续性影响。

第三，当代各国政治体系通常具备的六大制度元素，即构成现代代议政治框架的议会制度、选举制度、政党制度，以及行政制度（包括官僚政治）、司法制度和国家结构（单一制与联邦制）。这一部分大体涵盖了现代国家政治体系的三重基本关系，即：议会、选举、政党“三位一体”的代议原则；立法、行政、司法之间的权能划分；以及中央（或联邦）政府与地方政府之间的纵向分权。

第四，有关政治与社会之间的关联，大体处理为三大重要议题，即诸如利益集团、社会运动、公众舆论等形态的社会压力对政治生活的影响，各国之间政治文化的差异、变化及其对政治生活的影响，以及对政治变迁（政治发展）本身历史经验的对照和反思。

上述16讲内容当然不能完全涵盖当代政治学讨论的全部议题，尤其是基本没有专门讨论政治与经济、国家与个体、自由与平等之类重要的政治议题。通过上述内容的讲授与讨论，希望学员能够对于政治学的基本概念（尤其是基本概念之间的关系和张力）以及基本理论有所领略，从而能够举一反三，对更加深入的政治学或其他社会科学研究更加感兴趣（当然，首先是对政治世界感兴趣）。
