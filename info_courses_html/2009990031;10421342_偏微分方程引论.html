课程内容简介：
本课程主要介绍典型线性偏微分的物理意义、基本理论及基本求解方法。主要内容有：典型偏微分方程的建立，定解条件的物理意义；边值问题与分离变量法；波动与扩散及积分变换法；拉普拉斯方程的格林函数法；特殊函数简介及其在求解偏微分方程中的应用。
Course Description
This course introduces the physical derivations, fundamental theories and basic methods of contructing solutions for typical particial differential equations. It contains: the physical backgrounds of typical partial differential equations and well-posedness. Boundary value problems and separation of variables;Waves and difussions; Integration Transforms; Laplace’s Equation and Green’s Function;Introduction to special functions with applications in solving particial differential equations.
进度安排：

 

（零）预备内容 ﻿前两周：基础知识复习与预备：傅里叶级数，傅里叶变换，拉普拉斯变换基础知识和应用，多元微积分，常微分方程等复习与巩固．(4学时，可穿插在教学过程中)


 


（一）教学内容1（教师课堂教学学时（2小时） + 学生课后学习学时（≥4小时））


教学内容：绪论：三类方程的推导及定界条件，偏微分方程基本概念和基本知识，两个自变量二阶偏微分方程的分类


 


（二）教学内容2（教师课堂教学学时（6小时） + 学生课后学习学时（≥12小时））


教学内容：分离变量法：有界弦的自由振动，有限长杆的导热问题，二维Laplace方程边值问题，非齐次方程求解，具有非齐次边界条件的问题求解


 


（三）教学内容3（教师课堂教学学时（6小时） + 学生课后学习学时（≥12小时））


教学内容：行波法与积分变换法：达朗贝尔公式与波的传播，高维波动方程Cauchy问题（球面平均法和降维法简介），积分变换法


 


（四）教学内容4（教师课堂教学学时（4小时） + 学生课后学习学时（≥8小时））


教学内容：Green函数法： Green公式及应用，Green函数，Green函数的应用，试探法及Poisson方程求解


 


（五）教学内容5（教师课堂教学学时（10小时） + 学生课后学习学时（≥20小时））

教学内容：Bessel函数：Bessel方程及Bessel函数，递推公式，按Bessel函数展开为级数，Bessel函数的应用；勒让德多项式概论
