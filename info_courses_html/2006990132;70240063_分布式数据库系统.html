课程简介：
课程目标包括：1）巩固、加深数据库系统的基础知识，理解集中式数据库的不足与分布式数据库的优势；2) 掌握分布式数据库系统的基本原理和实现技术；3）了解当今分布式数据管理与应用中的前沿问题，包括并行与流数据管理、数据仓库与在线事务分析、NoSQL与NewSQL等；4) 构建一个分布式场景下的数据中心。 课程结束后，学生将1）熟知当前构建分布式数据库系统和服务的基本模型与实现方法；2）运用所学模型和方法解决具体问题；3）了解数据管理领域的研究方向和可能的研究结果；4）锻炼就某一问题展开研究的能力，包括查找相关资料、分析问题、解决问题，以及答案测试等；5）具备应用所学知识，解决一个实际大数据管理问题的能力。
英文课程简介：
Course Objectives: 1) to enhance the previous knowledge of database systems by deepening the understanding of the theoretical and practical aspects of the database technologies; showing the need for distributed database technology to tackle deficiencies of centralized database systems; 2) to introduce basic principles and implementation techniques of distributed database systems, including distributed database design and architecture, query processing and optimization, transaction management, recovery, and reliability protocols; 3) to expose active and emerging research issues in distributed database systems and application development, including parallel and streaming data management, data warehousing and OLAP, NoSQL and NewSQL; 4) to apply theory to practice by building a data center in a distributed context. After the course, the students can 1) get familiar with the currently available models, technologies for and approaches to building distributed database systems; 2) have developed practical skills in the use of these models and approaches to be able to select and apply the appropriate methods for a particular case; 3) be aware of the current research directions in the field and their possible outcomes; 4) be able to carry out research on a relevant topic, identify primary references, analyze them, and come up with meaningful conclusions; 5) be able to apply learned skills to solving a big data management problem.
