课程内容简介：
主要内容分为三个部分： 第一部分:智能机器人的基本概况，方法和工具。 1.1 智能移动机器人科技前沿 1.2 智能移动机器人硬件原理， 第二部分:基础准备， 2.1 智能移动机器人软件安装 2.2 Linux、ROS指令学习 第三部分：硬件学习 3.1 仿真机器人实践 3.2 机器人基本功能及指令学习 第四部分：实践应用 4.1 实现机器人自主巡航任务要求 4.2 了解课程竞赛规则 4.3 智能移动机器人自主巡航挑战赛/目标射击挑战赛
Course Description
The main content is divided into three parts: Part I: Basic overview, methods and tools of intelligent robots. 1.1 Frontier of intelligent mobile robot technology 1.2 Intelligent mobile robot hardware principle, Part II: Basic Preparation, 2.1 Intelligent mobile robot software installation 2.2 Linux and ROS instruction learning Part 3: Scene Building 3.1 Understand the Gazbo simulation environment 3.2 Create a virtual bot 3.3 Create a robot runtime environment 3.4 Functional construction of virtual robots Part IV: Practical Session: 4.1 Realize basic functions such as autonomous navigation of robots 4.2 Intelligent Mobile Robot Autonomous Cruising Challenge 4.3 Intelligent Mobile Robot Target Shooting Challenge 4.4 Physical Robotics Practice
