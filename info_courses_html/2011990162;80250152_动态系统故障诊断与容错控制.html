课程简介：
动态系统的故障诊断与容错控制发展概况；强跟踪滤波器（STF）理论及其在故障诊断中的应用；基于观测器理论的故障诊断方法；非线性系统故障诊断的未知输入观测器方法；基于神经元网络的故障诊断方法；线性系统的容错控制方法；非线性系统的自适应容错控制等。
英文课程简介：
In this course, we have opportunity to learn some basic definition of fault detection and diagnosis as well as fault tolerant control.
