课程内容简介：
质点力学，对称性与守恒定律，刚体的定轴转动，简谐振动，波动，狭义相对论基础，温度的微观意义，理想气体，热力学第一定律，卡诺循环，热力学第二定律，熵增加原理，卡诺定律。
Course Description
进度安排：

第一周 绪论 质点运动学：质点的运动函数、位移、速度、加速度、匀加速运动、抛体运动、圆周运动、相对运动。（自学：匀加速直线运动、SI单位和量纲，常见力、应用牛顿定律解题） 
第二周 牛顿运动定律: 牛顿运动定律、基本自然力、惯性系与非惯性系，惯性力、*科氏加速度与科氏力、*潮汐。动量与角动量:冲量与动量定理、质点系的动量、动量守恒定律。
第三周 火箭飞行原理、质心、质心运动定律、两体运动、质点的角动量和守恒定律、质点系的角动量和守恒定律。功和能: 功、动能定理、一对力的功、保守力、势能 。
第四周 功和能:万有引力势能、弹性势能、由势能求保守力、机械能守恒定律。刚体的定轴转动:刚体的运动、刚体的定轴转动定律、转动惯量的计算、刚体定轴转动定律的应用。
第五周 刚体的定轴转动: 转动中的功和能、刚体的角动量和角动量守恒定律、*进动。简谐振动的描述，旋转矢量与振动的相。
第六周 狭义相对论:牛顿相对性原理和伽里略变换、爱因斯坦相对性原理和光速不变。 
第七周 洛仑兹变换、时间膨胀、长度缩短、同时性的相对性, 相对论速度变换、相对论质量、相对论动量、相对论能量、相对论力的变换。

第八周 期中考试。期中总结。简谐振动的动力学方程，简谐振动实例，简谐振动的能量，阻尼振动、受迫振动。

第九周，共振，同方向同频率简谐振动的合成，谐振分析，相互垂直的简谐振动的合成。混沌。

第十周 机械波的形成。简谐波, 波动方程、波传播的能量，惠更斯原理，波的干涉，驻波，多普勒效应，声波，复波。 

第十一周 宏观与微观，温度，理想气体温标，水的三相点，理想气体状态方程。理想气体的压强和温度 。
第十二周 能量均分定律，麦克斯韦速率分布率；麦克斯韦速度分布率，麦克斯韦速率分布率的实验验证。
第十三周 玻尔兹曼分布定律，实际气体等温线，范德瓦耳斯方程，气体分子的平均自由程，*输运过程。  
第十四周 准静态过程，热力学第一定律，热容量，理想气体的等值过程，循环过程，卡诺循环，致冷循环 。
第十五周 自然过程的方向，热力学第二定律，热力学第二定律的统计意义 。热力学几率，玻尔兹曼熵公式，熵增加原理，可逆过程。
第十六周 卡诺定律，克劳修斯熵公式，熵的计算，温熵图。平衡相变简介，*耗散结构简介。
