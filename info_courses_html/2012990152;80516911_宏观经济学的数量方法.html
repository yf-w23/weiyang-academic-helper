课程简介：
这门课程面对博士二年级的希望从事宏观方面研究的学生。课程主要讲述如何编程求解常见的宏观模型，经过本课程的学习，学生应该可以独立求解常见宏观模型。课程内容包括宏观模型global solution和local solution的数量解法，以及如何计算heterogeneous agent/firm模型。 课程所授数量方法可以使用Matlab, Fortran, C, C++等编程语言进行实现，但是编程语言不是先修课程条件。
英文课程简介：
This course is intended to be suitable for second-year PhD student in macroeconomics. The aim of the course is to help the student to learn how to solve various models in macroeconomics. The course consists of three parts: the global solution method, the local solution method and the heterogeneous model solution. All sorts of programming languages can be used in computing the models, such as Matlab, Fortran, C and C++. However, programming language is not a prerequisite of this course.
