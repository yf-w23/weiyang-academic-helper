课程内容简介：
本课程旨在系统性地引导学生掌握智能机器狗的全栈开发流程。课程从搭建Python与Ubuntu开发环境、掌握机器人操作系统（ROS2）的核心通信机制（话题、服务、动作）入手，逐步深入到应用层面。学生将通过实践，依次实现基于传感器的自主避障、利用计算机视觉（OpenCV）进行目标检测与跟踪、以及运用SLAM技术完成激光雷达建图与自主导航。最终，课程将以一场综合性的团队挑战赛收官，要求学生集成所学全部技能，完成复杂的协同任务，从而将理论知识全面转化为工程实践能力。
Course Description
This course provides a systematic guide to the full-stack development of an intelligent robotic dog. It starts with the fundamentals, covering the setup of Python and Ubuntu environments and the mastery of core Robot Operating System (ROS2) communication mechanisms, including topics, services, and actions. Through hands-on practice, students will progressively implement key applications such as sensor-based autonomous obstacle avoidance, object detection and tracking using computer vision (OpenCV), and environment mapping and autonomous navigation with SLAM technology. The course culminates in a comprehensive team-based challenge, requiring students to integrate all acquired skills to complete a complex, collaborative mission, effectively transforming theoretical knowledge into practical engineering capabilities.
