课程内容简介：
本课程作为建环专业本科生接触专业课的第一学期的一门主要课程，秉承以“认识规律--学习方法--解决问题”的思路，向学生传授室内空气洁净技术的专业知识和方法。课程首先介绍颗粒物基本特性，然后着重介绍颗粒物的基本运动规律，在通过这些内容让学生“认识规律和原理”之后，以此为基础介绍空气过滤的基本原理和有关过滤器的结构和特性，从而让学生了解如何基于所认识和掌握的原理，发展解决问题的方法。进而引出空气洁净度等级和洁净室工作原理。然后着重介绍洁净室设计和计算的有关理论，并辅以实例说明。最后讲授洁净室的检测和评价方法，从而让学生认识如何全面、系统解决室内空气洁净的相关问题。 本课程安排一次洁净室的现场参观和一线工程师的讲解。课程有两次大型讨论课，一次是期中的课程作业开题答辩，一次是期末的课程作业答辩。此外，课程讲授过程中穿插5次课堂讨论。 本课程于2015年被评为清华大学本科生精品课程。
Course Description
进度安排：

第1讲 概述 主要内容：空气洁净技术的意义、内容，室内颗粒对人体、设备的危害和影响。从室内环境对颗粒物浓度控制的需求出发，介绍工业生产、人民生活对空气洁净技术的迫切需要，进而引起同学对于颗粒物的来源、危害以及主要控制方法的兴趣。 教学要素：讲授（LEC） 教学时数：2；实验时数：0；课外学时：2（查阅相关资料） 第2讲 室内颗粒的基本特征 主要内容：室内颗粒的形状、密度、大小以及粒径分布等。通过显微镜图片、扫描电镜图片等形式，展示颗粒物的形状、大小和粒径等，进而以实例说明室内颗粒物的浓度和粒径分布。 课堂讨论1：室内颗粒物的实测展示和讨论。 教学要素：讲授（LEC） 教学时数：2；实验时数：0；课外学时：2（作业、课外阅读：气溶胶领域的先驱事迹1） 第3讲 室内颗粒的运动规律 主要内容：从气溶胶力学的基础知识出发，介绍室内颗粒的受力、匀速直线运动、变速直线运动、曲线运动，引出颗粒物动力学的几个基本概念：空气动力直径、松弛时间以及惯性行程；进而介绍室内颗粒的扩散运动、沉积和凝并等，并针对室内环境介绍如何通过理论模型方法分析颗粒物的动力学要素对室内颗粒物浓度的影响； 课堂讨论2：颗粒物的沉积、穿透如何影响室内颗粒物浓度。 教学要素：讲授（LEC） 教学时数：4；实验时数：0；课外学时：4（作业和课外阅读：课外阅读：气溶胶领域的先驱事迹2） 第4讲 颗粒的过滤机理 主要内容：除尘方式概述，引出室内环境常用的过滤分离、纤维过滤器的过滤机理，进而介绍纤维过滤器的的过滤效率和影响纤维过滤器效率的因素，介绍经典的对数穿透定律和纤维过滤器填充率修正方法。 课堂讨论3：人肉净化器有用吗？ 教学要素：讲授（LEC） 教学时数：4；实验时数：0；课外学时：4（作业和课外阅读：课外阅读：气溶胶领域的先驱事迹3） 第5讲 空气过滤器的结构和特性 主要内容：空气过滤器的特性和分类、空气过滤器的特性指标，常见过滤器特性以及过滤效率的测试方法 课堂讨论4：如何自制净化/过滤器？ 课堂讨论5：口罩的作用？ 教学要素：讲授（LEC） 教学时数：4；实验时数：0；课外学时：4（课外阅读：为讨论课做准备） 中期开题报告 主要内容：学生按照分组，向全班报告所选项目大作业题目的背景、意义、技术路线和预期成果 教学要素：课堂讨论和问答互动 教学时数：6；实验时数：0；课外学时：6（查阅资料，与教师讨论，确定题目） 第6讲 空气洁净室基本概念和工作原理 主要内容：空气洁净度级别、洁净室原理（单向流、乱流以及矢流），生物洁净室原理及其压力控制要点 课堂讨论6：单向流洁净室可以“变向”吗？ 课堂讨论7：洁净室耗能的因素及可能的节能措施？ 教学要素：讲授（LEC） 教学时数：4；实验时数：0；课外学时：4（作业和课外阅读：洁净室形式和节能措施） 第7讲 空气洁净室设计计算理论 主要内容：洁净室均匀分布和不均匀分布计算理论。从颗粒物室内质量守恒出发，结合洁净通风系统，分别建立均匀和不均匀分布的计算方法。 课堂讨论：为什么洁净室设计计算可以忽略颗粒沉积和再悬浮？ 教学要素：讲授（LEC） 教学时数：2；实验时数：0；课外学时：4（作业） 第8讲 洁净室的设计计算 主要内容：从实际洁净室的设计计算出发，讲授必需的室内外计算参数确定、高效、中效以及有局部净化设备的净化系统设计计算方法。以5道例题讲述具体计算过程。 教学要素：讲授（LEC） 教学时数：2；实验时数：0；课外学时：2（作业） 洁净室现场参观 主要内容：学生分两组，到洁净室现场参观，与工程师交流洁净室的设计、运行和维护问题。 教学要素：现场参观和问答互动 教学时数：2；实验时数：0；课外学时：2（为参观做准备，准备与工程师交流的问题） 第9讲 局部洁净理论 主要内容：主流区的概念及其在局部洁净室中的应用和优势，主流区的速度、洁净度等特性，以及局部洁净区的具体应用形式。 课堂讨论：局部洁净的拓展？ 教学要素：讲授（LEC） 教学时数：2；实验时数：0；课外学时：2（作业和课外阅读：其它的局部洁净方式？） 第10讲 检测和评价方法 主要内容：采样理论和常用的采用方法，测定和评价中关于颗粒浓度、过滤效率和检漏、洁净室级别评定等问题，以及工程验收的一态、两段、第三方要点。 教学要素：讲授（LEC） 教学时数：2；实验时数：0；课外学时：2（作业和课外阅读：气溶胶 颗粒物检测设备） 期末课程论文答辩另行安排，6学时。等同于考试。
