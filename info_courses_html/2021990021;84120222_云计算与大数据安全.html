课程简介：
随着信息技术和人工智能的大规模应用，越来越多的服务被置于云平台和大数据分析平台等大型复杂计算系统中。云与大数据分析系统作为重要的信息基础设施，其安全性对于人们的生产和生活至关重要。本课程首先介绍云计算和大数据分析的基本概念，云与大数据分析系统的基本框架和基础技术，然后在此基础上全面讨论云系统与大数据分析系统的安全内容，包括云计算的安全、云存储的安全、云网络的安全，以及云与大数据分析系统中的数据安全与隐私保护。这些内容系统地涵盖了从用户和资源的身份验证，服务与API的访问控制，可信运行环境的构建（可信物理机、虚拟机和容器），安全的通信、会话与网络，文件和配置的完整性，到系统级的高可用性、应急响应机制、审计与合规性、隐私保护等云安全领域。在对云安全内容全面介绍的基础上，本课程还将讲授如虚拟技术的安全、根信任、密钥管理、安全启动等几个重点课题。学生在上完本课程后将具备扎实的云与大数据分析系统安全的基础知识，并熟悉它们和与它们类似的大型复杂计算系统的安全风险与防护技术，掌握其中一些重点安全内容的相应技术。
英文课程简介：
With large-scale adoption of information technology and artificial intelligence, more and more services are being placed in large complex computing systems such as cloud platforms and big data analytics platforms. Cloud and big data analytics systems are important information infrastructure now, and their security is crucial to people’s work and living. This course first introduces basic concepts in cloud computing and big data analytics, and basic frameworks and fundamental technologies for cloud systems and analytics systems. Based on them, the course then gives a comprehensive discussion of security for cloud systems and big data analytics systems, which overarches the security of cloud computing, cloud storage and cloud networks, respectively, as well as data security and privacy protection for cloud and big data analytics systems. These discussions cover cloud security areas systematically, from identification and authentication of users and resources, access control of services and APIs, construction of trusted runtime environments (including trusted physical machines, virtual machines, and containers), security of communications, sessions and networks, integrity of files and configurations, to system-wide high availability, incident management, audit and compliance, and privacy protection. On top of the overall introduction of cloud security, this course will also teach several key topics such as security of virtualization technologies, root trust, key management, and secure boot. Upon completion of this course, students will establish a solid foundation of knowledge on security for cloud and big data analytics systems, will get familiar with security risks and protection technologies for these systems as well as other similar large complex computing systems, and will be skilled in the technologies that deal with certain key issues of security for cloud and big data analytics systems.
