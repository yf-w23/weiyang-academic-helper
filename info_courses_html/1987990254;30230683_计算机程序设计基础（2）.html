课程内容简介：
通过本课程的学习，学生应该能掌握C++这门高效实用的程序语言和面向对象的程序设计方法，培养良好规范的编程风格；初步掌握常用非数值算法；进一步提高自己的编程能力、动手能力。
Course Description
Through the learning of this course, students should be able to master the efficient and practical programming language C++ and object-oriented programming method, cultivate good standardized programming style, grasp commonly used non-numerical algorithm, improve their programming ability and practical ability further.
