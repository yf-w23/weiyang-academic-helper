课程内容简介：
*本课程旨在通过研究型专题教学模式，帮助同学们系统、准确地掌握“毛泽东思想”和“中国特色社会主义理论”的基本观点及科学内涵，引导同学关注现实中的热点、重点、难点，自觉运用所学理论和方法去深刻理解“为什么”、积极思考“怎么办”，树立正确的人生观与价值观。 *课程内容涵盖：马克思主义中国化理论成果的精髓；毛泽东思想；社会主义建设道路的初步探索；中国特色社会主义基本经济制度、分配制度；党的领导等。 *本课程为2学分，共32学时；其中包括课内24学时（主要由教师进行课堂讲授）以及课外8学时（主要由同学们进行课下阅读）。
Course Description
进度安排：

课程导论；全球视野下的“共同富裕”；新自由主义经济政策的后果及对我国的启示；妇女解放与男女平等事业发展进程；医疗制度的百年演进；平台经济与高质量就业；课程回顾与总结
考核方式：
*考勤和参与15分：鼓励参与课堂讨论，缺席须通过邮件向教师请假。 *平时作业45分：从每个教学专题的课后思考题中任选三道作答。 *读书报告40分：以毛泽东同志经典名篇（指定）为基础撰写报告。
