课程内容简介：
本课程为清华大学风景园林本科专业基础课。主要学习大尺度区域与景观规划的概念、理论与方法的发展与演变，国际区域与景观规划代表性案例和技术等。本课程在三个层面开展学生培养，其中，在知识层面，使学生了解国内外区域与景观规划理论与实践的发展、演变与前沿动态，以及该领域代表性案例；在能力层面，使学生理解自然、文化与社会经济综合作用下的区域景观的类型、特征与机理，相应的景观规划的类型与内容，掌握规划程序与核心技术方法；在价值观方面，培养学生在区域与景观规划中秉持人与自然协调的正确价值观。
Course Description
进度安排：

第一讲：导论：区域景观规划的概念、理论、实践

第二讲：国家公园与保护地规划（+表述模型）

第三讲：景观生态规划（+评价模型）

第四讲：城市绿地系统规划（+变化模型）

第五讲：乡村景观规划（+实施模型）

第六讲：景观规划理论前沿（+反馈模型）

第七讲：景观规划实践前沿

第八讲：景观规划经典案例研讨
