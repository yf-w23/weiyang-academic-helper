课程内容简介：
本课程是高级语言程序设计的入门课程，完全针对零起点的学生，可作为其他信息类相关课程的基础课。目标是使学生通过本课程的学习，掌握面向对象程序设计的基本概念和方法、C++的基本语法和编程方法；学会使用集成开发环境；掌握程序调试方法。 课程提供可自行选择的考核方式：大作业或随堂测验二选一。
Course Description
进度安排：

 第1周

大课：第2章简程序设计（1）。

课外学时：下载阅读课程信息与教学要求，完成实验一，自学视频（雨课堂）第1章 绪论

第2周

大课：第2章简程序设计（2）。

实验课2学时+课外2学时：部分完成实验二和第2章编程作业。

第3周

大课：第3章函数（1）。

实验课：按照学校放假调课安排执行。完成实验二和第2章编程作业，可部分完成部分第3章实验和作业。

第4周

大课：第3章函数（2）。

      第4章类与对象（1）。

实验课2学时+课外2学时：继续实验三和第3章编程作业，可部分完成实验四和第4章编程作业。

第5周

大课：第4章类与对象（2）。

      大作业任务讲解

实验课2学时+课外2学时：完成实验四和第4章编程作业，做大作业。

第6周

大课：第5章数据的共享与保护。

实验课2学时+课外2学时：第5章实验和编程题作业，做大作业。

第7周

大课：第6章数组、指针与字符串（1）。

实验课2学时+课外2学时：部分完成实验六，做大作业。

第8周

大课：第6章数组、指针与字符串（2）。

实验课2学时+课外2学时：继续完成实验六和第6章编程作业，做大作业。

第9周

大课：第7章类的继承（1）。

实验课2学时+课外2学时：实验七和第7章编程作业，做大作业。

第10周 

大课：第7章类的继承（2）。

实验课2学时+课外2学时：继续完成实验七和第7章编程作业，做大作业。

第11周

大课：五一节放假

实验课2学时+课外2学时：上课按学校放假调课规定执行。

内容：复习已学过的所有章节，大作业初版完成。

第12周

原大课时间助教答疑（自愿参加），星期五第六节“随堂测验1”（课程冲突者可另选时间）。

实验课2学时+课外2学时：大作业助教指导与预答辩。

第13周

大课：第8章多态性（1）。

实验课2学时+课外2学时：部分完成实验八和第8章编程作业，做大作业。

第14周

大课：第8章多态性（2）。

实验课2学时+课外2学时：继续完成实验八和第8章编程作业，做大作业。

第15周

大课：第11章流类库与输入输出。

实验课2学时+课外2学时：继续完成实验八和第8章编程作业，提交大作业、准备大作业答辩。

第16周

随堂测验2 或 大作业答辩 。

通过课堂练习、实验、作业、大作业（也可不做大作业参加两次随堂测验）方式考核。
