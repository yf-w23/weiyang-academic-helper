课程内容简介：
本课程详细讲解数据库的基本原理、数据库的设计方法和利用数据库软件创建数据库的方法，旨在使学生掌握数据库系统的基本概念、数据库概念建模、逻辑建模的理论和方法、关系数据库标准语言SQL及其在现实中的应用。 主要内容分为两部分。第一部分是数据库原理，主要包括：ER数据模型、关系数据模型、关系代数、关系数据库的规范化理论和数据库设计方法等。第二部分是数据库应用，主要介绍结构化查询语言SQL的核心语句，开源数据库管理系统MySQL的主要功能，存储过程，嵌入式SQL的实现方法以及数据库事务管理的原理和方法。 本课程有助于培养同学们的数据素养。通过本课程可以掌握利用数据库的理论和技术，科学地获取、管理和应用数据的基本方法。具体来说，数据库设计阶段，通过概念模型和逻辑模型的设计和优化，避免出现数据冗余、不一致等更新异常问题。数据库实现阶段，通过各种技术方法保证数据的完整性和一致性。应用阶段，借助数据库系统实现业务自动化，如销售业务系统、财务系统、挂号系统、订票系统等等，也可以对业务中收集的数据进行汇总分析，为业务决策提供支持。
Course Description
This course combines classroom lectures with hands-on computer experiments and relates to real-life cases to explain in detail the basic principles of databases, database design methods, and the methods of creating databases using database software. It aims to enable students to master the basic concepts of database systems, the theories and methods of database conceptual modeling and logical modeling, the standard language SQL and its applications. The main content is divided into two parts. The first part is about database principles, mainly including: ER data model, relational data model, relational algebra, the normalization theory of relational databases, and database design methods, etc. The second part is about database applications, mainly introducing the core statements of the Structured Query Language (SQL), the main functions of the open-source database management system MySQL, stored procedures, embedded SQL, and the principles and methods of database transaction management. This course is helpful for cultivating students' data literacy. Through this course, students can master the basic methods of using the theories and technologies of databases to acquire, manage, and make use of data scientifically. Specifically, during the database design stage, through the design and optimization of conceptual and logical models, update anomalies such as data redundancy and inconsistent can be avoided. During the database implementation stage, the integrity and consistency of data can be ensured through various technical methods. During the application stage, by utilizing database systems, business automation can be achieved, as seen in sales business systems, financial systems, registration systems, ticket-booking systems, and so on. Additionally, these systems can aggregate and analyze the data collected during business operations to provide support for business decision-making.
