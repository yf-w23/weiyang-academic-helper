课程内容简介：
《信息光学》是光电信息类专业的核心课程之一，是信息技术与产业的基础，覆盖了电子信息、光学信息、计算机技术等领域。它是一门应用现代光学基本原理、思想方法来研究信息的获取、传递、存储、处理和显示，是对电子信息处理的进一步拓展。本课程主要以傅立叶变换、信号与系统、物理光学等理论为基础，阐明信息光学的原理和技术。可以帮助同学们理解光学的基本概念，包括对光波特性、成像系统和对光学仪器的理解等等。 1953年Zernike因为发明相衬显微镜以及1971年Gabor因为发明全息获得诺贝尔奖，都是信息光学领域的经典之作，并对半个多世纪以来信息光学的发展产生了深远的影响，是本课程的基本教学内容和案例。 随着VR/AR技术、自动驾驶、手机摄像、机器人视觉、无人机终端的爆发式发展，信息光学对新型光学成像系统的设计显得尤为重要，华为、字节跳动、小米、Meta、Google、Microsoft等公司的核心部门对光学工程师有着迫切的需求，信息光学课程是本专业方向的一门不可或缺的基础课程。 通过该课程学习可以熟悉现代光学的经典理论，基本概念。熟悉一些光学元器件，光学结构，光学系统的工作原理和实际应用。并了解智能光学技术与产业的过去，现在和将来，特别是国内外前沿的发展动态。课堂提供实验、参观、讨论、讲座等各种模式，培养大家在光学技术领域的分析、 创造和动手等研究能力。
Course Description
进度安排：

 

《信息光学》教学大纲

学分：2学分

学时：32学时（以2学时作为一个单元，共16个单元）

 

 

l  课堂授课：共8章（28学时，14次课）

第一章：绪论

1. 光学和光子学简介

2. 光学信息处理的发展趋势

3. 全息光学简介

第二章：傅里叶光学基础

1. 常用函数卷积与相关

2. 线性系统与线性空间不变

3. 抽样定理

第三章：标量衍射理论

1. 光波的数学描述和复振幅

2. 光波的角谱及角谱的传播

3. 标量衍射的角谱理论

4. 菲涅尔和夫琅和费衍射

第三章：光学成像系统的频率特性

1. 透镜的傅里叶变换性质

2. 衍射受限系统的相干传递函数和光学传递函数

3. 相干与非相干成像系统的比较

第四章：光学信息处理技术

1. 光学频谱分析系统和空间滤波系统

2. 相干光学信息处理

3. 非相干光学处理

第五章：全息原理及类型

1. 光学全息原理和基元全息图分析

2. 菲涅耳全息图和傅里叶变换全息图

3. 计算全息术和数字全息术

第六章：全息技术及应用

1. 全息光栅器件

2. 全息显示技术

3. 虚拟现实技术

第七章：光的调制器与探测器

1. 光的调制器

2. 光的探测器

第八章：先进显示与计算成像

1. 三维成像与三维显示技术

2. 计算成像与计算显示技术


 

l 数值计算实验（2学时）

通过数值仿真完成对图像的滤波处理，模拟光学信息处理中的空间频谱滤波过程。熟悉数字图像处理；熟悉Matlab编程；熟悉图像相关运算和卷积运算。

 

 

l 全息光学实验（2学时）

学习掌握光学信息处理的基本原理和实验技巧，了解体全息存储的基本原理和方法。

l 信息光学企业参观（2学时）

本课程提供到字节跳动、华为等合作单位的参观。
