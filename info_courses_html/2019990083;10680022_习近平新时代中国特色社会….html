习近平新时代中国特色社会主义思想概论

总学分：

课程内容简介：
本课程系统讲授习近平新时代中国特色社会主义思想的基本内容、精神实质、重大意义、实践要求等，引导学生深刻领会蕴含于以上内容之中的习近平新时代中国特色社会主义思想的世界观和方法论、立场观点方法，进而以此为指导正确认识和科学分析国内国际的重大社会现象和社会问题，逐步成长为担当民族复兴大任的时代新人。
Course Description
进度安排：

课程以教师课堂讲授为主，以教指委统编课件为依据，以“总体布局”为主要框架，融入党的二十大精神，结合清华具体情况，主要设置以下各讲：

1.《马克思主义中国化时代化新的飞跃》

2.《以中国式现代化全面推进中华民族伟大复兴》

3.《新时代中国特色社会主义经济建设》

4.《新时代中国特色社会主义政治建设》

5.《新时代中国特色社会主义文化建设》

6.《新时代中国特色社会主义社会建设》

7.《新时代中国特色社会主义生态文明建设》

8.《新时代坚持和发展中国特色社会主义的重要保障》
考核方式：
课程依据平时作业（25%）+读书报告（50%）+小班研讨（15%）+课程总结（10%）综合给分，并转化为等级制成绩。
