课程内容简介：
主要教授热力学和统计物理的基本原理，研究方法及其应用。注重基本原理与应用相结合，概念和应用并重。主要内容包括：热力学基本定律；均匀系统的热力学性质；相变热力学；近独立粒子系统的统计分布；Boltzmann统计理论；Bose统计与Fermi统计理论；系综理论(微正则系综，正则系综，巨正则系综）及其应用；非平衡统计的初步理论
Course Description
