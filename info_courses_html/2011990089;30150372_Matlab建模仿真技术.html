课程内容简介：
本课程为Matlab软件入门课程，围绕数学建模和系统仿真两大主题，介绍m语言编程、simulink/stateflow建模、AppDesigner编程、硬件在环仿真、自动代码生成技术等工程领域常用的软件技术。要求同学有C语言、线性代数（矩阵运算）和系统建模方面的基础知识。课程分为大班课堂授课和小班实验两部分。同学将掌握如何使用Matlab软件编写数据程序、建立系统仿真模型的能力，为未来学术科研和工程技术开发奠定基础。
Course Description
This course serves as an introductory program for MATLAB software, focusing on mathematical modeling and system simulation. It covers essential software techniques widely used in engineering, including M-language programming, Simulink/Stateflow modeling, App Designer development, hardware-in-the-loop (HIL) simulation, and automatic code generation. Prerequisites: Basic knowledge of C programming, linear algebra (matrix operations), and system modeling is required. The course consists of lectures (large-class teaching) and hands-on lab sessions (small-class tutorials). Upon completion, students will acquire the ability to write data processing programs and develop system simulation models using MATLAB, laying a solid foundation for future academic research and engineering development.
进度安排：

第一章：基于模型的电控系统开发方法概论

第二章：m语言编程基础及数据可视化

第三章：数学建模基础：数值计算、解方程

第四章：Simulink建模：普通模块、数学建模

第五章：Simulink建模：自定义模块、Simscape、Powertrain

第六章：Stateflow建模

第七章：AppDesigner编程

第八章：实时仿真、代码生成、软件管理（自学）
