课程内容简介：
可编程序控制器适用于恶劣的工业环境，80%以上的工业过程采用可编程序控制器系统控制。 本课程是典型的实践型教学课程，本课程介绍可编程控制器的历史、原理、国内外现状和发展趋势；介绍PLC与单片机、微机和继电器控制的区别；介绍IEC61131-3规定的PLC五种编程语言；结合实际工程案例讲解系统设计理念。介绍施耐德SCHNEIDER最新可编程控制器UnityPremium的硬件和软件编程；介绍流行组态软件Ifix\Citect\Kingview，介绍施耐德的最新变频器ATV-71及其应用系统设计。 课程依托清华大学-施耐德电气联合教学实验室，通过实际工业项目完成对学生的培养。所有实验项目不要求提交实验报告，通过项目实际运行来验收实验成果。 本课程对于电机系的同学可以作为电机系“专业任选课组”的课程。 “清华大学施耐德电气杯PLC系统设计大赛”就是依托本课程开展的。
Course Description
进度安排：

第一周：可编程控制器的历史、原理、国内外现状和发展趋势；

第二周：继电器逻辑控制系统；

第三-第五周：IEC61131-3规定的可编程控制器PLC五种编程语言；

第六-第八周：组态软件及变频器控制系统设计；

第九-第十周：可编程控制器控制系统设计；

第十一-第十五周：可编程控制器控制系统设计实验；

第十六周：综合测试。
