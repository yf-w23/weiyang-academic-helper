课程内容简介：
机器学习是一门研究如何让计算机无需显式编程就能有效工作的科学。在过去十年中，机器学习为人们带来了自动驾驶汽车、语音识别、机器翻译、人脸识别、网络搜索、推荐系统及对人类基因组更深入的理解。很多研究者认为机器学习是推进实现人工智能的最佳途径。本课程将系统地讲解机器学习，覆盖基本原理、经典模型、高效算法与应用实践。内容包括：1. 监督学习（广义线性模型、支持向量机、决策树、Boosting）；2. 无监督学习（聚类、量化、降维、稀疏编码）；3. 强化学习（MDP、蒙特卡洛、时间差分、值函数近似、策略梯度、演员评委模型）；4. 学习理论（PAC学习、泛化界、偏差方差分解）。本课程将通过数学推导和编程实践，帮助学生使用机器学习技术解决大数据和人工智能中的具体应用问题。
Course Description
Machine learning is the science of getting computers to act effectively without being explicitly programmed. In the past decade, machine learning has given us self-driving cars, speech recognition, machine translation, face recognition, web search, recommender system, and in-depth understanding of the human genome. Machine learning is widely believed to be the best approach to advancing human-level AI. This course will provide a systematic introduction to machine learning, spanning fundamental principles, classical models, efficient algorithms, and practical applications. Topics include: (1) Supervised learning (generalized linear models, support vector machines, decision trees, boosting); (2) Unsupervised learning (clustering, quantization, dimensionality reduction, sparse coding); (3) Reinforcement learning (Markov decision process, Monte Carlo, temporal difference, value function approximation, policy gradient, actor-critic); (4) Learning theory (PAC learning, generalization bound, and bias-variance decomposition). The course will dive into mathematics derivations and programming practices to help students learn how to apply learning algorithms to real problems in big data and artificial intelligence.
