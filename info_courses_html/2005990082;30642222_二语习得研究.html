课程内容简介：
本课程将阐述关于二语言习得本质的一些基本问题。在整个课程中，我们将学习二语习得的过程、理论和研究方法、并运用所学的分析一系列的问题，如母语习得方式和理论、二语习得方式和理、影响二语习得的各种因素，等等。木课程的主要内容包括：二语习得概述、母语习得研究、二语习得研究、中介语、课堂教学、个性差异和二语习得、社会文化因素和二语习得、二语习得课题和数据。
Course Description
This course will provide some answers to basic questions about the nature of second language acquisition (SLA). Throughout the course, we will examine a number of ways in which a second language is learned and researched and use what we have learned to address a variety of questions—including how the first language is acquired, how a second language is learned, what roles instruction plays in SLA, what factors affect SLA, and so on. This course mainly covers the following topics: overview of SLA, the study of first language acquisition, the study of SLA, interlanguage, instructed second language learning, individual differences and SLA, social-cultural factors and SLA, research topics and types of data in SLA.
