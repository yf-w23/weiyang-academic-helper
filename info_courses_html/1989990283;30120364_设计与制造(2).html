课程内容简介：
本课程结合前沿制造技术和科研成果，系统全面介绍现代制造工程中主要制造技术和制造工程设计的基本原理与方法，主要内容有：金属成形制造原理与工艺（铸造，压力加工，焊接）、金属切削加工原理与工艺（金属切削原理，机床、刀具、夹具设计原理，加工表面质量和精度），非金属制品的制造，精密特种加工，以及进行制造工程技术组织的工艺规程设计原理。本课程强调理工融合、设计制造融合、冷热加工融合，并安排切削仿真与数控加工的专题研究。通过本课程学习，能够系统掌握各种制造技术的基础理论及其之间的内在联系，培养创新意识和分析解决制造工程实际问题的能力。
Course Description
进度安排：

绪论（1学时）：制造的概念；课程主要内容、目的、学习方法及要求

第1单元：金属成形制造（20学时）

1.1    金属铸造

1.2    金属塑性成形

1.3    焊接

第2单元：非金属制品的制造（4学时）

2.1 塑料成形

2.2 橡胶制品加工

2.3 玻璃制品成形

第3单元：金属切削加工（24学时）

3.1 金属切削原理

3.2 机床与刀具

3.3 工件的定位夹紧

3.3 机械加工精度与表面质量

第4单元：精密与特种加工（4学时）

4.1 超声加工

4.2 激光加工

第5单元：工艺规程设计（12学时）

5.1 机械加工工艺规程设计

5.2 装配工艺规程设计

 

课程实验：机床与刀具基础实验；组合夹具的设计和组装实验

专题研究：切削仿真与数控加工
