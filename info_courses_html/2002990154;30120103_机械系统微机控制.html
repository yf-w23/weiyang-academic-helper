课程内容简介：
以最新的科研成果为基础，以具体的工程应用为背景，以书面授课和实验研究相结合的方式，培养学生学习并掌握机械系统微机控制的发展趋势、关键技术、硬件系统和软件设计等基本知识及其典型应用。 课程重点讲授机械系统微机控制基本原理、单片机（MCU）的基本原理、主要外设功能和程序设计，以及可编程控制器（PLC）的工作原理、硬件结构、程序设计和组态应用编程等。 实验内容以工程需求为背景，以实际工程应用为目标，开设多组基础型、提高型和创新型单片机和PLC控制软硬件实验，努力践行研究型实践教学模式。
Course Description
进度安排：

第1周：绪论、机械系统微机控制概述

第4周：单片机I/O实验

第5周：STM32单片机的中断功能与实验

第6周：五一放假

第7周：STM32单片机的TIM与实验

第 8周：单片机的A/D功能与实验

第 9周：单片机综合实验（选作）

第10周：PLC基础知识

第11周：西门子S7-200PLC系统特性及硬件组成

第12周：S7-200编程语言

第13周：STEP7编程及I/O实验

第14周：S7-200触摸屏及wincc组态软件

第15周：PLC组态软件应用及实验

第16周：PLC电梯综合实验
