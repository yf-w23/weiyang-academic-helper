课程简介：
大数据给数据分析带来了前所未有的挑战，机器学习是处理大数据的重要手段，而最优化算法广泛应用于科学与工程计算、数据科学、机器学习、人工智能、图像和信号处理、金融和经济、管理科学等众多领域，是机器学习的核心基础。本课程基于机器学习，介绍其中一些典型的最优化问题的建模，最优化理论，以及最新的优化算法。主要讲授（1）机器学习中的优化模型：LASSO及变形、基追踪问题、低秩矩阵恢复、逻辑回归、图像恢复等等；（2）最优化基础-凸分析：共轭函数、次梯度、适当闭函数性质等等；（3）各种机器学习优化模型的最优性理论：不可微无约束优化、非凸约束优化、复合优化等问题的最优性理论；（4）无约束优化算法及收敛性分析：次梯度算法、有限内存BFGS等等；（5）约束优化算法及收敛性分析：LASSO问题的二次罚函数法、基追踪问题的增广拉格朗日函数法；（6）复合优化算法及收敛性分析：LASSO问题的近似点梯度法、Nesterov加速算法、近似点算法、对偶近似点梯度法、ADMM，逻辑回归的随机梯度下降算法等。
英文课程简介：
Big data brought unprecedented challenges to the data analysis, machine learning is one of the important means to deal with large data, and optimization algorithms are widely used in science and engineering calculation, data science, machine learning, artificial intelligence, image and signal processing, the financial and economic, management science, and many other fields, is at the core of machine learning. Based on machine learning, this course introduces modeling, optimization theory, and the latest optimization algorithms for some of these typical optimization problems. (1) Optimization models in machine learning: LASSO and deformation, basis pursuit problem, low rank matrix recovery, logistic regression, image recovery, etc. (2) optimization basic-convex analysis: conjugate function, subgradient, suitably closed function properties, etc.; (3) The optimality theory of various machine learning optimization models: the optimality theory of non-differentiable unconstrained optimization, non-convex constrained optimization, composite optimization and other problems; (4) Unconstrained optimization algorithm and convergence analysis: sub-gradient algorithm, finite memory BFGS, etc.; (5) Constrained optimization algorithm and convergence analysis: quadratic penalty function method for LASSO problem, augmented Lagrange function method for basis tracking problem; (6) Composite optimization algorithm and convergence analysis: LASSO problem approximate point gradient method, Nesterov acceleration algorithm, approximate point algorithm, dual approximate point gradient method, ADMM, logistic regression stochastic gradient descent algorithm, etc.
