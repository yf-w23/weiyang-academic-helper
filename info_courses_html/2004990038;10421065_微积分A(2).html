课程内容简介：
本课程内容包括：n维欧氏空间中的点集拓扑，多元函数的极限和连续，多元函数的偏导数与全微分、方向导数、梯度，向量值函数的微分，高阶偏导数，隐函数定理，Taylor公式，极值问题；含参定积分和含参广义积分；重积分概念、性质、计算及应用；两类曲线曲面积分及三大公式：Green公式、Gauss公式、Stokes公式，向量场初步；数项级数、函数项级数、幂级数、Fourier级数。
Course Description
Course Description: point set topology in n-dimensional Euclidian spaces; limit and continuity of multi-variable functions; partial derivatives and total differentials, directional derivatives, gradients, vector value functions: differentials and chain rules, implicit function theorem, Taylor's formula, maximum/minimum value with constraints; multi-integrals, iterated integrals, variable substitution, applications to geometry and physics, line integrals and surface integrals, Preliminary vector fields, Green formula, Gauss formula and Stokes Formula; number series, function series, power series, Fourier series，integrals with parameters.
进度安排：

 微积分A(2)教学安排：

第一周：n维欧氏空间中的点集拓扑及多元函数的概念；多元函数的极限与连续

第二周：多元函数的偏导数与微分；方向导数与梯度

第三周：映射与微分；隐函数微分法

第四周：多元函数微分学几何应用——空间曲线与曲面；多元函数的极值

第五周：条件极值；含参定积分

第六周：含参广义积分；二重积分的概念及性质

第七周：二重积分的计算；三重积分的计算

第八周：重积分应用；曲线积分

第九周：第二型曲线积分性质及第二型曲线积分的计算；

第十周：格林公式；全微分方程与第一型曲面积分

第十一周：五一调休

第十二周：第二型曲面积分与Gauss公式、Stokes公式

第十三周：场论初步；数项级数

第十四周：正项级数敛散性判别；函数项级数

第十五周：函数项级数和函数的分析性；幂级数和函数的性质

第十六周：Fourier级数；Fourier级数的性质
