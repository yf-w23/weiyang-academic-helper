课程内容简介：
课程以Python语言作为工具，注重讲解计算机程序设计的概念、方法和思路，培养学生的问题分析、算法设计和编码能力。主要内容包括：顺序结构程序设计、选择结构程序设计、循环控制、字符串、函数、数据集合、图形用户界面和递归算法等。课程每周授课3学时，另有3个学时的课外上机实验。
Course Description
The course uses the Python language as the programming tool and focuses on the concepts, methods and ideas of computer programming. We emphasise on improving the students' problem analysis, algorithm design and coding skills. The main content consists of: sequential program structure, selection program structure, loop, string, function, data collection, GUI and recursive functions. There will be a three-hour lecture and a three-hour lab.
进度安排：

 

第2周：第1章概述（数值类型、算术运算与表达式、输入与输出、R进制）

第3周：第2章简单程序设计（顺序结构程序设计、选择结构程序设计）

第4周：第3章循环控制（for语句、while语句、break和continu语句）

第5周：第3章循环控制（编程举例）

第6周：第4章字符串类型（字符串概述、Python字符串、程序举例）

第7周：第5章函数（函数概述、函数的使用、函数调用的实现过程）

第8周：第6章数据集合（数据集合概述、列表）

第9周：第6章数据集合（元组、集合、字典）

第10周：作业讲评（编程规范、程序调试、作业样例）

第11周：第7章图形用户界面（绘制主窗口、绘制图形、事件处理、动画）

第12周：第8章面向对象程序设计（类和对象、类的定义和使用、类的基础）

第13周：第9章递归算法（递归概述、基于分治的递归算法、基于回溯的递归算法）

第14周：第10章Python标准库（内置函数、文件与字节流、模块）

第15周：课程复习

第16周：课堂小测试
