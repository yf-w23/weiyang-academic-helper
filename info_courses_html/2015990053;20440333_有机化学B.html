课程内容简介：
本课程对有机化合物的的结构、性质及反应机理进行系统介绍，对各类化合物的化学反应、制备和工业生产、用途等进行较为系统的论述，着重对重要的有机反应机理的学习和讨论。
Course Description

一、有机化学基础 （6课时）

二、烷烃（2课时）

三、立体化学基础（1课时）

四、卤代烃及亲核取代反应（3课时）

五、烯烃和消除反应（3课时）

六、烯烃的加成反应（3课时）

七、炔烃和共轭二烯（3课时）

八、醇 酚 醚 （3课时）

九、芳香化合物（3课时）

十、醛和酮（3课时）

十一、羧酸及其衍生物（3课时）

十二、烯醇及烯醇负离子（3课时）

十三、胺类化合物（3课时）
