课程内容简介：
“写作与沟通”课程（简称“写作课”）是清华通识教育体系的重要组成部分。课程主要目标是培养选课学生的逻辑思维和批判性思维，提升选课学生的写作与沟通能力。写作课将以主题教学的模式展开，依靠小班研讨、高强度的阅读、师生一对一面批以及朋辈交流研讨的模式，帮助学生展开与文献的对话，进行批判性思考，形成有创造性的观点，最终按照学术规范的要求进行写作。 在主题式写作课中，每个课堂都围绕特定的主题展开，选课学生可以从不同主题的课堂中选择自己感兴趣的一门。“无专业门槛、有学理深度”的写作课主题设置，将保证来自不同专业、不同学科背景的学生均可开展有深度的研究，完成准学术性说理文的写作，形成突破学科壁垒的逻辑思维和批判性思维。 写作课采用小班制教学的方式，班级规模为16人左右，以保证教师与学生之间、学生与学生之间形成积极互动、思维碰撞的氛围。课程将利用文本研读、资料分析、研讨交流、观点展示等各种环节训练同学的思维、阅读、写作与沟通能力。 特别提示：写作与沟通课不同的课堂有各自的主题，请去“进度安排”中阅读主题介绍等相关信息，避免误选。
Course Description
进度安排：

本课程主题：游戏与文化

2026春季学期课序号：96、97、98、99

主题简介： 

游戏并非人类独有的行为，但游戏对于人类有着非凡的意义。哲学家柏拉图把神圣等同于游戏，美学家席勒将“与美游戏”看作是人的最完满状态，文化史学者赫伊津哈干脆提出人类的文明就是在游戏中产生发展而来。另一方面，游戏在历史和现实中也从不缺少争议和贬斥。“业精于勤而荒于嬉”揭示了游戏在普遍价值观念中的负面形象，形形色色的“洪水猛兽”论则暴露了几十年来中国社会面对游戏时的深度焦虑。

游戏为何会引发如此两极的评价？关于游戏，我们有哪些洞见，哪些偏见，又有哪些“未见”？立足于具体经验和观察，我们可以提出更多的问题：数字游戏的兴起是否彻底改写了游戏的内涵？虚拟与现实的边界的模糊，是“侵蚀”还是“开创”？作为大众文化商品，游戏带来的是新的文化生命力和意义空间，还是一场瓦解深度价值的消费迷梦？游戏赋予了我们新的交往可能，还是加深了个体原子化的状态？游戏已经是或者正在孕育新的艺术形式，还是已有艺术的“新瓶装旧酒”，甚至会导致“艺术的终结”？“游戏化”是解放的力量，还是操控的技术？在“数字劳动”“平台经济”的语境下，游戏扮演着什么样的角色？甚至可以追问，当今的游戏是在实现着游戏的本质价值，还是在践踏自身的潜能？

以上这些宏观问题不会有一个唯一的答案，我们也不应满足于对它们的泛泛空谈。我们可以从某一个具体的游戏事实或案例入手，通过深入观察与思考，从中发现待解的问题并给出我们自己的回答，从而参与到这场极为重要的时代论争当中。

期待同学们从文化分析的视角切入游戏这个主题，通过经验关照、知识探索、理性思辨和学理论说，在某个具体而微的着眼点上给出自己的精彩回答，并在其中实现视野的打开和自我的重塑。
