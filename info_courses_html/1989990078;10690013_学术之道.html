课程内容简介：
为进一步落实“清华学堂人才培养计划”（以下简称“学堂计划”）的“领跑者”理念，加强文化素质教育，邀请学堂班各班首席教授开设“学术之道”课程
Course Description
进度安排：

本课程大班讲授全学期15次，其中首席教授报告共12次。其他主要教学环节包括：小班讨论及导修15次，课后经典阅读，写作等。本课程要求学生听取首席教授报告，仔细阅读指定材料，积极参与小组发言与讨论，完成2000字左右的命题写作和5000字以上的自选题课程论文各1篇。
考核方式：
课程评分标准如下：命题写作和课程论文（50%），讨论环节（40%），出勤（10%）
教材及参考书：
每位首席教授负责两次报告，同时提供课后阅读材料。阅读量建议每次课后至少2学时，约80—100页，英文可以酌减，古文可再酌减
