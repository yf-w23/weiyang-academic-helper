课程内容简介：
随着互联网、万维网、在线社交网络的发展，网络科学和技术得以全面发展，形成一门新兴的 交叉学科。本课程全面系统地介绍当代网络科学的各个方面，包括小世界模型、无标度网络与 优先连接模型、网络中心性、网络鲁棒性、网络信息与影响力传播、网络社区检测等。本课程 侧重网络科学的数学理论及复杂网络中的算法设计，将理论基础、算法设计与网络科学之应用 相结合。
Course Description
With the rapid growth of the Internet, World Wide Web, and online social networks, network science and technologies emerge and develop into a new interdisciplinary subject. This course will systematically cover various aspects of modern network science, including the small-world model, scale-free networks and preferential attachment, network centralities, network robustness, information and influence propagation in social networks, network community detection, etc. This course emphasizes the mathematical theory supporting network science and algorithm design for complex networks. It aims at a balanced integration between theory foundation, algorithm design, and applications of network science.
