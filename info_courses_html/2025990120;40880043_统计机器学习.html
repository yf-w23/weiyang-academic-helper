课程内容简介：
“统计机器学习” 是一门以统计理论为核心支撑、衔接数据与价值的基础课程，旨在帮助学生掌握从数据中挖掘规律、构建预测与分析模型的核心逻辑。课程整体围绕 “统计思想→模型构建→实践应用” 展开，覆盖经典模型与现代算法。课程从统计决策理论基础出发，重点讲解有监督学习的经典模型(线性模型、支持向量机、树模型与集成学习等)，然后讲解无监督学习的统计推断方法(包括经典的聚类与降维方法，以及社区发现算法)；同时以业界实例进一步构建从课堂理论理解到实际模型应用完整学习链，同时体会算法在实际场景中“四两拨千斤”的巨大作用。
Course Description
"Statistical Machine Learning" is a foundational course based on statistical theory. Its main goal is to help students equip students with the fundamental logic of discovering patterns from data and building predictive and analytical models. Structured around the framework of "statistical thinking → model construction → practical application," the course covers both classical models and modern algorithms. Starting with the basics of statistical decision theory, it focuses on key supervised learning models (including linear models, support vector machines, tree-based models, and ensemble learning), followed by the statistical inference methods in unsupervised learning (including classical clustering and dimensionality reduction algorithms). With real industry cases, the course further builds a complete learning chain—from understanding classroom theories to applying models in practice—while allowing students to appreciate how algorithms can deliver transformative impact with elegant efficiency in real-world scenarios.
进度安排：
  • Lec 01. 03/02: 课程介绍；Statistical Decision Theory (了解机器学习概况，建立有监督学习的整体框架: 基于损失函数学习模型参数，理解极大似然和交叉熵损失的等价性)
  • Lec 02. 03/29: Regression: OLS, Subset Selection, Shrinkage (回顾经典线性回归，通过从一元线性回归到多元线性回归的算法了解QR分解；能够通过分块矩阵求逆给出intercept和slop part的分别表示，并体会特征中心化对于线性回归的影响)
  • Lec 03. 03/16: Regression: Ridge, Lasso及通过LARS求解 (理解Ridge/Lasso的不同形式: 加约束项和目标函数加正则，以此了解广泛应用的正则化思想；学习Ridge regression和Lasso的基本求解方式)；Classification Evaluation (学习分类问题的简单评估指标，了解需要根据场景特点选择评估方式)
  • Lec 04. 03/23: Classification Evaluation: ROC curve, AUC (学习AUC的概率含义及定义)；Classification Baisic Models: LR, Naive Bayes, KNN (学习分类的基本模型；明确LR是Neural Network的特例，通过代码加强认知；了解简单的NN模型构建与训练)
  • Lec 05. 03/30: Support Vector Machine (I): Optimal Hyper Plane, Soft margin Hyperplane (学习线性SVM, 体会建模过程: 从直观想法转化为优化问题、并且根据数据需求调整模型设定；学习对偶问题定义、KKT条件)
  • Lec 06. 04/06: 假期愉快！(04/04-04/06放假)
  • Lec 07. 04/13: Support Vector Machine (II): Nonlinear SVM., Kernel Function, RKHS. (学习非线性SVM, 通过代码实例体会从线性到非线性时模型表达能力的变化；初步了解Kernel方法与再生核希尔伯特空间，并且通过基于核方法定义的MMD距离及其在图像风格迁移任务中的应用，体会抽象的数学理论对于实际问题的巨大帮助)
  • Lec 08. 04/20: 期中考试 
  • Lec 09. 04/27: Decision Tree: ID3, C4.5, CART. (理解树模型的本质: 递归划分和贪心求解，深入了解CART算法细节: 包括嵌套剪枝的理论基础，能够不借助package, 通过代码从零实现决策树)
  • Lec 10. 05/04: (04/30-05/06放假，本次课调整至05/09周六上课，请注意时间～) Ensemble Learning (I): Bagging (including Random Forest) and Boosting Algorithm (AdaBoost). (了解集成学习的思想以及基本范式，通过经典论文回顾感受集成学习对于算法表现的显著提升，并且能够直观理解算法背后的motivation)
  • Lec 11. 05/11:  Ensemble Learning (II): Gradient Boosting (GBDT); XGBoost. (学习更加现代的集成学习算法，了解XGBoost在tablular数据中相较于NN的优势地位，理解其背后原理并且了解xgboost package的基本用法和扩展选项).
  • Lec 12. 05/18: Uplift Modeling (brief introduction to Causal Inference). (理解Causation和Association的不同, 通过具体案例理解uplift modeling和一般有监督学习的差异，以此为起点了解causal inference的主流框架；学习uplift的评估方式及基本建模方法；并且通过coupon pricing的实际案例，了解互联网场景下建模的基本流程，体会通过算法模型解决问题的思考过程).
  • Lec 13. 05/25: Unsupervised Learning (I): Dimension Reduction. PCA/ SNE (t-SNE)/UMAP. (了解无监督学习和有监督学习的差异；学习基本的线性降维方法PCA及其背后原理，了解主流的非线性降维方法SNE和UMAP及其应用场景；能够借助已有package实现算法满足需求).
  • Lec 14. 06/01: Unsupervised Learning (II): Clustering (K-means, Hierarchical Clustring) 以及 经典的社区发现算法 (网络数据的聚类). 以及 期末报告Presentation (如果最后一节课时间不够，会在本次课进行一部分)
  • Lec 15. 06/08: 期末报告Presentation. (06/13开始期末考试周)
