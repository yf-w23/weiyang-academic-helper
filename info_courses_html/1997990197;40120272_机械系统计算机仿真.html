机械系统计算机仿真

课程内容简介：
介绍计算机仿真的基本概念、进展与工程应用。讲授连续系统仿真的原理、建模方法和核心算法，用连续系统仿真方法建立机械系统的系统仿真模型，分析机械系统的动态性能。讲授机械系统动力学和运动学的建模与仿真分析，讨论应用仿真优化机械系统设计的案例。介绍离散事件系统仿真的原理、建模方法和算法，讨论离散制造系统的仿真分析案例。
Course Description
进度安排：

计算机仿真的基本概念，发展及应用；连续系统的建模方法；一般机械系统的连续系统仿真；连续系统仿真的核心算法；基于MATLAB的一般机械系统仿真；多刚体机构的动力学和运动学仿真；机构设计与仿真分析的案例；离散事件系统仿真的原理和算法；离散事件系统的建模与分析方法；离散制造系统仿真分析的案例。
