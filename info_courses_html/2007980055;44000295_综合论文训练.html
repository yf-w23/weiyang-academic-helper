课程内容简介：
综合科研训练是培养学生运用所掌握的理论知识和技能，在所选导师指导下进行专门科学研究、培养创新能力的重要环节，也是培养学生良好的科学素养和合作精神的重要机会。基本要求是在第四学年春季学期开始至春季学期结束，其中用2周完成论文开题报告，并专门从事10周实验室实验活动。此训练活动将有机结合自二年级春季学期开始的实验室轮转活动以及进一步开展的为期两年的学生志趣实践活动，以保证学生有足够的时间完成一定工作量的创新活动。
Course Description
进度安排：

第七学期末,院（系）审查学生完成课程学习情况。开始综合论文训练准备工作，进行课程动员，学生与指导教师联系，开始选题及前期准备。申请跨院系进行毕设的学生需提交申请表，院系及教务备案。

第八学期第1-2周,进行综合论文训练课题统计，填写相关信息（课题名称及类别；指导教师姓名、职称、指导人数等；学号、学生姓名、班号；课题来源）。

第八学期第3-4周,正式开题：学生做开题报告（含文献综述、方案论证、进程计划等，并撰写外文资料的调研阅读报告或书面翻译），并记成绩。（指导教师评分）

第八学期第8-9周,中期检查：学生每人口头汇报、解答问题，并记成绩。系教学管理部门组织检查，对不合格的学生给予警告。检查日程通知教务处，教务处、督导组随机抽查。（4~5名教授参与）

第八学期第14-15周,终期答辩：学生提交论文，完成论文查重工作，提交论文查重工作总结。完成论文答辩：学生将论文按要求装订成册，答辩前由指导教师、评阅教师写出评语，答辩后由答辩小组给出成绩。必须有系级抽查答辩，并检查评分标准执行情况，不符合标准的，由教学副系主任（副院长）及系（院）教学主管部门进行调整平衡，教务处、督导组随机抽查。优秀论文评选按照院里流程。

第八学期第16周之前,完成综合论文训练评分、优秀论文评选等工作。
