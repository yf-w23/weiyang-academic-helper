课程名：
声乐基础提高
总学时：

总学分：

课程内容简介：
1、讲述声乐基本理论知识，发声特点，变声期嗓音的训练和保健有关知识。 2、基本能以正确的呼吸方法进行歌唱。 3、在巩固中声区的基础上适当扩展音域��发声要求连贯、流畅、圆润。 4、歌唱语言清楚��字正腔圆。 5、力求用学到的发声技术��完整的表达歌曲艺术和思想。
Course Description

考试
