课程简介：
本课程主要关注信息系统安全，通过本门课程，学生可以进行信息系统安全攻防初步实践，并具备安全系统设计的基本能力。主要课程内容如下： 1、信息安全概论-背景、应用、意义（4课时） 2、信息安全概论-基本定义、组成要素（6课时） 3、Web渗透攻防理论与实践-SQL注入（2课时） 4、Web渗透攻防理论与实践-漏洞分析与工业网络安全（2课时） 5、Web渗透攻防理论与实践-大数据下的网络身份安全认证（2课时） 6、信息安全体系的新思想-面向领域的安全（2课时） 7、信息安全体系的新思想-重视安全管理（4课时） 8、信息安全体系的新思想-用系统的视角看问题（4课时） 9、系统安全实践-案例分析（4课时）
英文课程简介：
This course focuses on information system security. Through this course, students can conduct preliminary practice about information system security and defense, and have the basic ability of designing security system. The course includes 1.Information Security: background, application and significance 2.Information Security: basic definition, constituent elements 3.Theory and practice of web penetration attack and defense: SQL injection 4.Theory and practice of web penetration attack and defense: vulnerability analysis and Industrial network security. 5.Theory and practice of web penetration attack and defense: network identity security authentication under large data 6.New ideas of information security system: safety of a specific domain 7.New ideas of information security system: safety management 8.New ideas of information security system: system view 9.System security practice: Case study
