课程内容简介：
本课程是机械设计基础系列课程的主要组成部分，旨在以通用机械零部件的设计为载体，介绍机械典型零部件的工作能力与结构设计的一般方法；从而使大家进一步掌握机械设计的基本方法、基本理论和基本思路。 主要内容包括： 一、总论（机电产品设计的一般过程、工程材料、国家标准等基础知识介绍） 二、机械系统的工作能力设计 1)机械传动零部件的工作能力设计； 2)机械支承零部件的工作能力设计； 3)机械联接的工作能力设计； 4)计算机辅助设计方法的典型应用。 三、机械系统的结构设计 1)轴系的结构设计； 2)机械联接的结构设计； 3)其它结构设计等。 课程的主要教学环节包括：讲课、讨论课、大作业和实验。
Course Description
进度安排：
第1周：总论（机械设计的基本知识、课程描述）；
第2~8周：机械传动零件的工作能力设计（齿轮传动、蜗杆传动、带传动的工作能力和相关结构设计）；
第9~15周：轴系零部件的工作能力设计（轴毂联接、轴、滑动轴承设计、滚动轴承选型及滚动轴承支撑结构设计）；
第16周：螺栓连接的设计
