课程内容简介：
电路原理C课程包括3大部分内容，即电阻电路分析，动态电路的时域分析和正弦激励下动态电路的稳态分析。各部分的主要内容分述如下。 1. 电阻电路分析。 （1）基本概念和元件：电压，电流，功率，电阻，独立电源，受控电源，金属氧化物半导体场效应管（MOSFET），运算放大器，二端口网络。 （2）分析方法：基尔霍夫定律，电阻等效变换，电源等效变换，回路法，节点法。 （3）电路定理：叠加定理、替代定理、戴维南定理和诺顿定理。 （4）非线性电阻电路的方程列写和求解：图解法、分段线性法和小信号分析方法。 2. 动态电路的时域分析。 （1）基本概念和元件：电容，电感，线性动态电路，零输入，零状态，一阶电路，二阶电路，状态方程。 （2）分析方法：一阶电路的三要素法，二阶电路的分析方法，状态方程的列写方法。 3. 正弦激励下动态电路的稳态分析 （1）基本概念和元件：相量，频率响应，滤波器，互感，变压器，理想变压器，谐振，有功功率，无功功率，复功率，视在功率 （2）分析方法：相量法，对称三相电路的抽单相法，周期性非正弦激励作用下动态电路的稳态分析法
Course Description
Principles of Electric Circuits is one of the kernel courses in the broad EECS subjects. Almost all the required courses in EECS are based on the concepts learned in this course, so it’s the gateway to a qualified EECS engineer. The main content of this course contains linear and nonlinear resistive circuits, time domain analysis of the dynamic circuits, and the steady state analysis of the dynamic circuits with sinusoidal excitations. Important concepts, e.g. filters, resonance, quiescent point, etc., cutting-edge elements, e.g. MOSFETs and Op Amps, etc., systematic analyzing tools, e.g. node method and phasor method, etc., and real-world engineering applications, e.g. square wave generator and pulse power supply for railgun, etc., will be discussed in depth.
