课程内容简介：
注意：该课程提供双语、全英文两个课堂选择。 本课程主要面向信息系统、信息管理、商务分析以及其他需要程序设计技能的专业的本科生。学生将学习不同计算机语言（如C、Python）的基本概念及程序设计的基础知识，并进行实际的程序设计训练。学生通过本课程的学习，将掌握运用各种不同的计算机语言进行程序设计处理计算问题的能力。 具体的内容包含（但不限于）： (1) 编程语言的基本概念 (2) 数据类型 (3) 变量 (4) 操作符 (5) 控制流 (6) 函数 (7) 数组 (8) 结构体 (9) 指针 (10) 文本处理 (11) 输入输出 本课程的具体教学目标包括： (1) 学生能够掌握多种计算机语言的基本概念； (2) 学生能够运用多种计算机语言进行程序设计； (3) 学生能够培养一种利用程序设计处理问题的思维模式。
Course Description
Note: The course offers two language choices: Bilingual vs. English. This course is for undergraduate students who plan to have a career in areas like information systems, information management, business analytics and other areas that require programming skills. Students will learn the basic concepts of various different computer languages (e.g., C, Python) and the basic knowledge of programming, and have actual programming practices. After this course, students are expected to use different computer languages for programming to solve computational problems. Specific topic includes (not limited to): (1) concepts of programming languages (2) data types (3) variables (4) operators (5) control flow (6) functions (7) arrays (8) structure (9) pointers (10) text processing (11) input and output The goals of this course are: (1) students will understand the concepts of computer languages; (2) students will be able to write programs using different computer languages; (3) students will be able to solve computational problems using programming.
