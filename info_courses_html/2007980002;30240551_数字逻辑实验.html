课程内容简介：
作为计算机人才培养中不可或缺的部分，数字逻辑实验是第一门专业基础数字电路硬件实验课程，具有实践性强、工程性和系统性强的特点。通过本课程的学习，使学生能够正确观察和分析实验现象，掌握基本实验方法，培养基本数字电路实验技能，通过运用所学知识，设计制作较为复杂的数字电路，培养学生数字逻辑设计与综合应用的能力、使用计算机辅助设计工具的能力，全面提高学生的素质、动手能力和创新能力。 实验内容有两类： （1）中小规模器件实验：与非门电路的测试、简单组合逻辑电路的设计等。 （2）大规模FPGA实验：点亮数字人生、加法器设计、计数器设计、密码锁设计等。
Course Description
进度安排：

本课程是实验为主的课程，重在锻炼动手设计能力，历时16周。

第一周介绍具体实验进度安排、实验方法和System Verilog语言基础，单独约同学们一个晚上的课堂时间讲授为主；

三至六周为示波器使用及小规模器件电路实验；

期中后直到十五周均以FPGA和综合实验为主；

第十六周考试。

此安排若遇到法定节假日将顺延。
考核方式：
平时实验、示波器实验测试、综合设计考核，按等级制记成绩
