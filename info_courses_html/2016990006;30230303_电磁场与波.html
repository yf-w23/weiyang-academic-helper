课程名：
电磁场与波
总学时：
48
总学分：
3
课程内容简介：
讲授电磁场的基本定律，麦克斯韦方程组，静电场，标量电势，静电场问题的求解，恒定磁场，矢量磁势，标量磁势，时变电磁场，波动方程，坡印廷定理，能量和能流，平面电磁波，反射与折射，波导与谐振腔，时变场的标势与矢势，达朗伯方程，偶极辐射，天线辐射
Course Description
进度安排：

第一章：电磁现象的基本定律（8学时）

1.1  电磁场的基本实验定律。

1.2  矢量场和矢量函数，矢量函数的微分和积分

1.3  麦克斯韦的位移电流学说。积分形式和微分形式的麦克斯韦方程组。

第二章：静电场（20学时）

2.1  静电场的标量势(位)函数，标量泊松方程与拉普拉斯方程。

2.2  静电标量势函数的多极展开，电偶极子

2.3  静电场中的导体。

2.4  媒质的极化,电介质，电感应强度，介电率，本构方程。

2.5  静电场和静电势的边界条件。

2.6  静电场的能量和储能密度。

2.7  拉普拉斯方程的边值问题。

2.8  唯一性定理。

2.9 分离变量法，矩坐标系，圆柱坐标系，球坐标系。

2.10      镜像法。

2.11      格林函数。

2.12      恒定电流场

第三章：恒定磁场（6学时）

3.1 恒定磁场的矢量势(位)函数，库仑规范，矢量泊松方程。

3.2 静磁场磁矢势的多极展开，磁偶极子。

3.3 媒质的磁化，磁场强度，磁导率，磁性材料。

3.4 磁场的能量和储能密度。

3.5 恒定磁场的标量势函数。

第四章：时变电磁场与电磁波的传播（18学时）

4.1 时变电磁场，瞬时值形式和复数形式的麦克斯韦方程。

4.2 波动方程和亥姆霍兹方程。

4.3 时变电磁场的边界条件。

4.4 时变电磁场的能量和能流，坡印亭定理。

4.5 均匀平面电磁波。

4.6 导电媒质中的平面波，良导体与完纯(理想)导体。

4.7 平面波在介质界面上的反射与折射，布儒斯特角，临界角与全反射。

4.8 平面波在导体面上的反射。

4.9 波导。

4.10 电磁谐振器。

4.11 介质的色散。

第五章：电磁波的辐射（12学时）

5.1 时变电磁场的动态势(位)函数，洛伦兹规范，达朗伯方程。

5.2 达朗伯方程的基本解，推迟势。

5.3 李纳－维谢尔势。

5.4 动态势的多极展开。

5.5 电偶极辐射，短天线，半波天线。

5.6 磁偶极辐射。

5.6 荷电粒子的辐射。
