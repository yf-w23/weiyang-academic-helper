课程内容简介：
《微机电系统设计》课程主要讲授微机电系统中基本结构的设计方法、微机电器件的设计方法、以及在设计过程中涉及到的电学、力学、热学和生物学知识。课程围绕微机电系统的三大类器件：微传感器、微执行器和微流体控制器，重点介绍的微机电基础元件包括：（1）薄膜；（2）导电通道；（3）隔膜；（4）应变电阻；（5）微梁；（6）毛细管（微流体通道）；（7）压电元件；（8）双压电元件；（9）热效应元件；（10）气动元件。课程重点讲授各种基础元件的工作原理、基础元件的机、电设计方法、基础元件的机-电信号转换、提取与激励、以及多种元件组合的集成设计方法。
Course Description
进度安排：
微机电器件的主要功能是实现传感功能，或者执行特定动作，也就是我们通常所说的“微传感器”和“微执行器”。其功能实现的主要途径是通过变形，当外界因素引起微机电元件的变形时，这种变形可以由一个或几个元件的组合转换为电信号，实现对外界的传感功能；当微机电元件在自身电信号的激励下产生变形时，这种变形可以对外界施加力或者位移，实现特定的执行动作。
       通过本课程的讲授，希望同学们能够理解和掌握如下几个方面的重点内容：
（1）    各种微机电基础元件的工作原理和设计要素；
（2）    各种微机电基础元件的变形分析方法和结构优化方法；
（3）    各种微机电基础元件的机-电信号转换、提取或激励方法。
（4）    微流体的运行规律与控制方法；
（5）    多种基础元件组合的集成设计方法；
（6）    利用有限元分析软件进行微机电系统设计。

本课程为48学时，3学分。共16讲，每讲为3学时。
