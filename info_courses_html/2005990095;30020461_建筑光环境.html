课程内容简介：
系统讲解建筑光环境的设计原理和设计方法，内容包括天然光环境和人工光环境的视觉评价、设计标准、计算方法、应用实例与美学问题等。课程对光度学、色度学、视觉的基本概念和光气候照明设备等基本知识也有全面的介绍。
Course Description
进度安排：
系统讲解建筑光环境的设计原理和设计方法，内容包括：
（1）建筑光环境概论。综述学科概况与研究热点，光与生命、文化、建筑科学的关联。
（2）视觉概论。介绍视觉系统、视觉任务/能力/能见度特征、视觉环境评价方法。
（3）色度学概论。介绍各种色度体系，浅析建筑色彩规划与设计。
（4）光度学基础。介绍光度学各项指标及其测试方法，并结合实例进行梳理。
（5）照明计算。介绍简化算法，灯具性能，结合实例进行计算与选灯训练。
（6）绿色照明。介绍绿色照明的理念、实践与研究。
（7）天然光环境设计的历史与理论、标准与计算方法、最新研究成果、实践案例。
