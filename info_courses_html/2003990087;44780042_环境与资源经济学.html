课程内容简介：
《环境与资源经济学》是一门面向书院本科生的综合性专业基础课程，以理解资源与环境约束下社会经济系统的运行机制，并探索可持续发展路径为主线，引导学生认识和理解资源与环境问题的社会经济系统成因，并在此基础上探索具有可持续性与现实可行性的解决路径。课程从广义资源与环境经济学视角出发，重点关注自然资源与环境容量作为稀缺约束条件，如何在经济社会系统中影响发展路径、技术选择与制度安排。课程以气候变化与碳中和等现实议题为主要情境载体，引导学生分析碳排放空间、生态承载力等关键资源约束所引发的跨期选择、代际公平与公共治理问题，体现当代环境与资源经济学由“分部门问题”向“系统性约束分析”的拓展。
Course Description
进度安排：

本课程围绕“资源与环境约束下的社会经济系统如何运行，以及如何设计可持续的治理与发展路径”这一主线，设置三个相互衔接的教学模块：

模块一：资源与环境约束下的决策基础（1-8周）
关键词：外部性、公共物品、成本效益、跨期选择
• 资源与环境问题为何具有系统性与公共性？
• 私人决策与社会最优之间的差距从何而来？
• 成本效益分析在现实决策中如何被使用，又存在哪些局限？
目标：建立对资源与环境决策中权衡、约束与取舍问题的基本认知。

模块二：支持资源与环境保护的治理工具箱（9-14周）
关键词：激励机制、政策工具、行为偏差、制度设计
• 为什么在存在明显收益的情况下，部分资源保护与减排措施仍难以实施？
• 不同治理工具（碳税、补贴、碳市场等）分别适用于哪些情境？
• 行为偏差与制度设计如何影响政策效果？
目标：理解不同治理工具的作用机制及其适用边界。

模块三：资源、环境与经济发展的动态关系（15-16周）
关键词：创新、结构转型、发展路径、协同效应
• 资源约束条件下，环境治理是经济发展的负担还是转型机遇？
• 技术创新如何改变资源配置与减排成本结构？
• 不同发展阶段面临怎样的路径选择与政策空间？
目标：跳出“环保 vs 发展”的二元对立，理解长期结构性转型问题。
