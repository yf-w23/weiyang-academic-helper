课程内容简介：
本课程是一门本科生选修课，为有兴趣了解如何在现代计算系统上提升程序性能的本科生开设。 从智能手机到现代处理器、图形处理器，再到世界上最大的超级计算机，并行计算无处不在。充分利用这些计算引擎，需要了解和掌握并行程序方法与技术，也需要了解关键的计算硬件性能特征。本课程的目标是深入理解在现代并行计算系统上设计程序所涉及的基本原理和工程权衡，重点讲授有效利用现代计算系统所必需的并行编程技术、性能分析方法与工具，同时涵盖硬件设计以及它如何影响软件和算法的设计。
Course Description
进度安排：

教学内容按照基础架构、并行编程、性能分析与优化的顺序组织授课，辅以案例与工具讲解，便于同学们建立感性认识，由浅入深，条理化学习。

主要教学环节：课堂教学32学时，课后复习，适当扩展阅读，动手实验并完成作业和课程项目。

课堂讨论：课上组织至少1次课堂讨论，请同学们交流课程项目选题和进展情况，并随时提供课程答疑。

课程内容包括：并行计算概述、优化串行程序性能、并行程序模型与并行编程原则、多线程编程与性能优化、多进程编程与性能优化、GPU编程与性能优化、并行程序性能分析与建模。

 
考核方式：
不安排期中和期末考试。根据平时出勤情况，作业、课程项目完成质量综合评定成绩。
