课程内容简介：
面向对象已成为计算机领域的主流技术之一，课程将以C++ 语言为基础，详细介绍用C ++进行面向对象的程序设计的核心思想。C++ 语言设计成熟、理论体系完整，全面支持面向对象设计思想，应用十分广泛。以C++ 进行面向对象程序设计，可以保证程序的高度可复用性、可维护性，有利于团队协作，便于复杂系统的设计与实现。 本课程是计算机科学的重要基础课程，要求学生在已掌握的计算机语言程序设计的基础上，能进一步拓展到面向对象的C++ 编程。课程内容衔接前序课程《程序设计基础》和后续课程《数据结构》和《软件工程》。课程重点讲解面向对象程序和大规模程序的设计方法论问题，通过讲解C++基础知识、面向对象的设计思想、原则和方法，并通过实践项目真正实践面向对象编程的理论与方法。 通过本课程的学习，要求学生全面掌握用C++ 语言进行面向对象程序设计的基本概念和方法，建立面向对象的思维模式，并能熟练地用C++ 实现这些思路。具体内容包括：C++ 面向对象的基本语法、面向对象的编程方法和思想，以及设计模式等内容。要求学生理解对象、继承、多态、模板等编程技巧。通过完成规定的设计练习，加强和提高程序调试的水平和能力。通过完成大作业，掌握有一定规模程序的设计方法，培养团队协作的能力。
Course Description
This course will introduce key features of Object-Oriented Programming (OOP) and typical design patterns based on the OOP features using C++, which is widely used both in industry and scientific research projects. C++ is a powerful multi-paradigm programming language, which supports procedural programming, objected-oriented programming, generic programming, etc. Using C++ and the design patterns based on OOP, we can design large projects, which guarantee easy code reuse and good adaptability to future change (extension) in response to user requirements. This course is an advanced course aiming at further improving the students' capability both in programming skills and program-designing skills, which requires the basic programming skills particularly in C. Main contents of the course are as follows: C++ class, object, inheritance, polymorphism, template, STL, and design patterns.
进度安排：

第1周：编程环境，主要介绍多文件编译、MAKE工具、命令行参数等。

第2周：类、抽象、封装、对象。

第3周：构造函数、析构函数、函数重载、缺省参数。

第4周：宏、常数、静态。

第5周：引用、拷贝构造函数。

第6周：流、操作符、组合、继承。

第7-8周: 虚函数和多态。

第9-14周：设计模式、模板等高级主题。

第15-16周：分组展示报告与点评。
