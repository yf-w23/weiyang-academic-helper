课程内容简介：
社会经济发展需要什么样的人才？应该选择什么样的人进入政府管理社会？这些现代社会治理的关键问题都离不开对教育制度的理性思考。本门课程围绕教育与社会选拔、教育体系的社会学分析、以及全球化和教育变革三大主题，讲述人类知识的演化、社会分析的概念体系、教育社会学的学科属性、社会研究的方法、社会不平等与分层、教育与社会流动、性别与教育平等、班级社会学、学校组织社会学、教师职业社会学以及全球化与教育变革共十一讲内容。课程回应了清华学生关于“寒门能否出贵子”、“是否选择教育为职业”、“教育制度改革的公平和正义”等现实困惑，帮助学生认识自我、教育和社会的关系，从而进一步发展并建构自我身份的认同。
Course Description
进度安排：

第一部分： 导论

第一讲 课程导论

    讲述人类知识的演化（宗教、形而上、科学），让学生从人类知识生产和传递（教育）的角度理解教育和社会、个体和教育的关系。

第二讲 什么是社会学？

    讲述社会学核心概念和理论分析框架，比较社会学和其他社会科学的异同（社会唯名和唯实论） 

第三讲 什么是教育社会学？

    讲述教育社会学的学科缘起、演变和发展，以及在中国的引入与传播

第四讲 访谈法：一种社会研究方法

研讨课（一）清华本科生的学习体验：以数学课为例

 

  第二部分：教育与社会不平等

第五讲 社会性不平等

    讲述社会性不平等的概念以及社会分化的理论

第六讲 教育与社会流动 

    讲述教育的社会分化和社会流动功能，以及相关理论

研讨课（二）什么是性别？如何认识性别？

 

第三部分：教育活动的社会学分析

第七讲 班级体系的社会学分析

   讲述作为社会体系的班级角色分化和习得

第八讲 学校组织的社会学分析

   讲述组织理论视野中的学校（分类、制度化和潜课程） 

第九讲 教师职业的社会学分析

   分析职业的社会性特征、以及教师职业的专业化特征

 研讨课（三）什么是自我？如何维系、发展自我？

             

第四部分：全球化与教育变革

第十讲 全球化与教育变革

   讲述全球化概念、指标和测量以及教育全球化的趋势和对中国的挑战

研讨课（四）：什么是教育？教育于我、于社会何用？
