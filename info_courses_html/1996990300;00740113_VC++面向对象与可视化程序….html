课程内容简介：
本课程讲述面向对象与可视化程序设计的思想和方法。内容涉及Visual C++中的基于MFC的编程。重点介绍面向对象程序设计中的事件驱动消息响应的机制和利用MFC构架编写应用程序。涵盖程序开发界面中常用的文档、菜单、工具条、滚动条、对话框、按钮、编辑框等对象，以及字体、鼠标、键盘、光标、动画等应用和多媒体的应用开发。为在教学过程中提高学生的应用开发能力，本课程针对每个知识点配有一个综合实例开发作为授课的例子，通过实例让学生全面掌握“面向对象与可视化程序设计”的思路、开发技巧与体系。选此课程的学生，要求先修C语言程序设计相关课程。
Course Description
This course introduces the ideas and methods of object-oriented and visual programming. The content covers MFC-based programming in Visual C++. This couesr focuses on the mechanism of event-driven message response in object-oriented programming and the use of MFC architecture to write applications. It covers the development of documents, menus, toolbars, scrollbars, dialog boxes, buttons, editing boxes and other objects commonly used in the program development interface, as well as applications such as fonts, mice, keyboards, cursors, animations, and multimedia. In order to improve students' application development ability in the teaching process, this course is equipped with a comprehensive case development for each knowledge point as an example of teaching, so that students can fully grasp the ideas, development skills and systems of "object-oriented and visual programming". Students who take this course are required to take C programming related courses.
