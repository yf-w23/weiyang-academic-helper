课程简介：
本课程在突出基础性、逻辑性和理论性的同时，以一种统一的框架来论述辨识的理论和方法，从较高的层次阐述系统辨识的实质和内在联系。课程主要内容分四部分：（1）辨识系统与模型及实验设计，包括系统的时域描述和频域描述、线性时不变/时变模型、非线性模型、开环辨识信息实验、开环可辨识性条件、闭环辨识信息实验、闭环可辨识性条件、D优化准则、Fisher信息矩阵的性质；（2）辨识方法及其数值计算，包括最小二乘类辨识方法与AUDI算法结构及其UD分解实现、梯度校正辨识方法、极大似然辨识方法、预报误差辨识方法、多变量系统辨识方法、闭环系统辨识方法、模型阶次辨识方法；（3）辨识理论与性能分析，包括递推辨识算法的统一框架、递推辨识算法的ODE收敛性分析、最小二乘类辨识算法的收敛性条件及收敛速度分析；（4）辨识应用与实际考虑，包括系统分析与可辨识性判断、辨识偏差的影响因素分析、准则函数与模型类选择、模型质量分析与模型检验、辨识软件包框架、辨识的实际应用等。
英文课程简介：
This course introduces the theory and methodology of system identification in a unified framework. This course is offered to graduates and includes the following topics: (1) systems, models, and experiment design; (2) system identification methods and numerical computation; (3) system identification theory and performance analysis; and (4) system identification applications and practical issues.
