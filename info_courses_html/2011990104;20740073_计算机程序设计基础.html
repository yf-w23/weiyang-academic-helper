课程内容简介：
该课程以C语言程序设计内容为基础，注重讲解程序设计的概念、方法和思路，培养同学的基本编程能力，以及逻辑思维和抽象思维能力。
Course Description
进度安排：
第1讲：课程介绍、计算机基础原理、编程环境准备（3课时）
介绍教学目标、学习方法、考核方式，编程工具。介绍计算机组成原理，发展过程以及计算机语言的分类和发展。
 
第2讲：C语言概述（3课时）
C语言简史，C的特点，C语言的基本概念，用5个实例给大家简历C程序的基本概念和组成，包括基础主函数、函数、语句、基本单词、输入输出函数，整形和浮点型，返回值。同时介绍作业自动评测系统。
 
第3讲：数据类型、运算符和表达式（3课时）
数据类型，常量与变量的声明与使用（变量声明的意义），整型数据与实型数据的定义与使用，操作符与表达式（操作符与表达式在语言的表现以及相互关系），赋值，输入输出与格式化输入输出。
 
第4讲：控制流（3课时）
语句与程序块；if、if-else、else-if语句；switch语句；选择结构的嵌套与组织，goto语句的历史地位与问题，for、while循环程序设计。
 
第5讲：函数与程序结构1（3课时）
函数的定义与意义，函数调用，函数参数与返回值，函数原型，函数的嵌套调用，函数通信方法，局部变量与全局变量，变量的存储类；
 
第6讲：函数与程序结构2（3课时）
程序块结构；递归函数，递归调用时的栈框架，一般递归策略与递归信任。
 
第7讲：库与接口设计（3课时）
编译预处理与条件编译，多文件组织，函数库的组织与结构，标准库。
 
第8讲：程序Debug与编程技巧（3课时）
程序常见错误；高效编程的技巧； 基本函数设计原则，数据封装与信息隐藏。
 
第9讲：数组（3课时）
一维与二维数组的定义与使用，字符数组的定义与使用；
 
第10讲：指针1（3课时）
指针的基本概念，指针与地址，指针与数据对象，指针的定义与使用，指针与变量；
 
第11讲：指针2（3课时）
指针运算，宜使用指针的场合与不宜使用指针的场合。指针与多维数组的定义与基本使用方法；
 
第12讲：输入与输出（3课时）
标准输入、输出；格式化输出；变长参数表；格式化输入；文件访问；错误处理；行输入和行输出；其它函数；
 
第13讲：文件（3课时）
文件描述符，文件数据结构，文件结构体与文件指针，文件的访问（打开、读写、定位、关闭）。
 
第14讲：算法设计与分析（3课时）
算法的基本概念与特征，算法的描述方法，算法的设计与实现，算法分析与算法复杂度，常用算法设计与分析。
 
第15讲：数据抽象1（3课时）
抽象数据类型，线性表，栈与队列的基本概念，栈与队列的应用。
 
第16讲：数据抽象2（3课时）

符号表与哈希表，抽象符号表及其应用。
