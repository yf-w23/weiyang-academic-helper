课程内容简介：
课程内容包括细胞的结构、功能和细胞（个体）的生命活动。涉及的主要实验技术有：显微镜操作技术，染色技术，动物细胞培养技术（无菌操作），实验动物操作技术，等。 实验项目分为基础实验和拓展实验两个模块。实验项目如下： 1、基础实验 显微镜操作，植物原生质体的制备，细胞融合诱导及观察，HeLa细胞传代培养，HeLa细胞凋亡诱导及检测，细胞转染，乳鼠肾细胞的原代培养，HE染色。 2、拓展实验（三选一） rRNA原位杂交，免疫荧光染色，RNAi（可能会根据具体情况增开1-2个拓展实验项目）。
Course Description
进度安排：

实验一（第3周）、绪论课；正置普通光学显微镜操作及操作考试1（正置显微镜的使用等）；  

实验二（第4周）、培养细胞的HE染色及观察；倒置相差显微镜及数码成像系统；

实验三（第5周）、动物细胞培养及相关技术，HeLa细胞传代培养及生长周期观察；

实验四（第6周）、乳鼠肾细胞原代培养； 

实验五（第7周）、细胞转染及观察；荧光显微镜操作&confocal操作

实验六（第8周）、HeLa细胞凋亡检测；流式细胞分析仪操作

实验七（第9周）、植物原生质体制备及融合； 

实验八（第12周）、随堂测试（“笔试”），操作考试2（细胞传代、倒置显微镜的使用）；  

实验九（第12周等）、 拓展选做实验：三选一，（1）rRNA原位杂交-第12周周六（全天）&周日（半天），（2）免疫荧光染色技术-第12周（分为两组进行，分别安排在周五&六或周六进行实验），（3）RNAi-3.19 19:20 绪论课； 3.26 // 4.2 //4.16…… 19:20讨论课，每组根据项目进展情况，自行安排操作及截止时间） 。 

总结（第14周）、实验报告讲评、总结等（“拓展选择实验展示”）。  

 注：

（1）转染实验包括转染操作（约1.3小时，拟安排在第7周上课日期前一天进行，具体时间根据同学们的课程安排确定。）和结果观察（第7周上课时间，分组进行，每组约1.5小时）。（时间安排有冲突的同学，可调整至其他时间进行操作）   

（2）传代操作考试中传代的细胞用于拓展实验1和2，故拓展实验1和2将安排在第12周的晚上或（和）周末进行；  

（3）细胞生长周期观察和原代培养观察需利用课外时间进行，期间细胞间将定时开放，每位同学根据所培养的细胞的状况进行观察和记录。

（4）每项拓展选做实验上课时间不同，多数项目需在2个或以上时段中完成，一般在晚上或者周末开展实验。 
考核方式：
考核方式为考查。其中：实验报告占总成绩的55%，操作考试1占5%，操作考试2占总成绩的15%，课堂表现（操作、实验习惯、等）占总成绩的20%，随堂小测（2次）占总成绩的5%。 成绩按等级制录入。
