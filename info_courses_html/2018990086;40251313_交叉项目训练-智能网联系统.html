课程内容简介：
本课程主要内容包括以下内容： （1）智能网联系统的基础知识及体系架构 （2）节点软硬件架构 （3）网络构建技术 （4）应用层技术，包含数据管理、定位、同步、系统优化决策技术。 本课程包括课程实验与实践环节，其中包括智能网联系统底层编程、节点通信、无线组网、网络安全、分布式系统设计等技术的应用，并最终要求学生设计完成一套具有基本功能的智能网联系统。
Course Description
The contents of this course include: (1) The fundamental and architecture of intelligent network system; (2) The software and hardware for the endpoints; (3) The network stack; (4) Application Layer Technology, including: data management, localization, synchronization, and system optimization. A series of lab sessions is designed to provide hand-on experience of the students. The lab sessions cover the following topics: endpoints programming, networking, wireless ad-hoc networks, information and network security, distributed system design. The students are expected to leverage all the technologies learned to create a functional intelligent network system prototype.
进度安排：

1. 智能网联系统绪论

2. 节点层：操作系统1, 实验：树莓派网络配置和远程访问

3. 节点层：操作系统2，实验：树莓派传感器与执行器操作

4. 节点层：传感器，实验：树莓派多线程编程

5. 物理层：调制解调，实验：多树莓派有线组网

6. 物理层：无线通讯，实验：多树莓派打地鼠游戏实现

7. 链路层：隐终端、暴露终端问题，实验：基于Batman协议的多树莓派无线自组网

8. 链路层：介质访问控制协议，实验：基于无线自组网的多树莓派打地鼠游戏实现

9. 安全：对称加密、消息认证码，实验：Wireguard VPN

10. 安全：非对称加密，TLS，实验：基于QUIC协议的云、端节点架构

11. 系统服务：时间同步、定位、大数据、云计算

12. 实验：基于树莓派的节点定位与时间同步

13. 综合应用：智能交通、智能建筑，实验：基于容器技术的边缘树莓派节点服务部署

14. 实验：基于K8s的容器编排技术的多树莓派管理

15、16. 综合课程设计、课程设计答辩
