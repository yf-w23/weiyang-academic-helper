课程内容简介：
本课程教授飞行基础知识和理论，通过飞行实践培养学生基本飞行能力和技术。课程的基础理论包括：飞行原理、飞机系统、民航法规、领航学、航空气象学、航空生理与医学、人的因素、机场与空域、航空无线电与空管、航空仪表、机务维修、飞行前准备、起落航线、空域动作、特情处理等；实践教学部分包括：1）飞行模拟器实验，通过飞行模拟器完成起落航线、空域动作和转场飞行；2）无人机飞行实验，操纵无人机完成视距内和视距外飞行；3）热气球飞行实验，操纵热气球起落；4）参观通用航空机场（航站楼、机库、塔台），观摩初级固定翼有人机飞行（起落航线和空域动作）。通过本课程学习，不仅让学生掌握飞行必要的基础理论知识，而且建立对主流飞行器的认知。为进一步学习飞行和飞行器设计奠定理论和实践基础。本课程填补了清华大学航空系飞行技术类课程空白，是已有课程体系的有效补充，健全了学科课程体系。培养学生飞行热情和飞行能力，提高学生国防和航空报国意识。
Course Description
进度安排：

 教学计划包括4个月，共计16周。

第一个月，课程教学，包括：飞行原理、系统仪表、无线电通信与空管、领航学、飞行准备与空域动作等；

第二个月，飞行实验，包括：飞行仪表、6自由度平台模拟飞行、720度平台模拟飞行、空管、攻防演练等；

第三个月，课堂教学，包括：人的因素、气象学、机场空域、民航法规、特情处理、飞行计划等；

第四个月，实验与实践，包括：无人机飞行实验、热气球放飞实验、机场参观。
考核方式：
理论课考核：随堂测验，50%成绩； 实验课考核：模拟器飞行考核，无人机操控考核，50%成绩
