课程内容简介：
本课程立足于西方音乐剧（musical theater）的学术领域，以二十世纪西方社会文化为重要背景，讲述音乐剧“形成-发展-成熟-辉煌-裂变-多元化发展”的百年历程，旨在剖析音乐剧百年历程中所呈现出来的风格演变及其背后的历史根因。 通过本课程的教学，学生将： 1.了解西方音乐剧产生、发展、成型的历史脉络； 2.了解不同西方音乐剧流派产生的历史、宗教、社会等文化根因； 3.了解西方音乐剧的主要风格流派、代表人物和典型性作品； 4.了解音乐剧作为一门独立的舞台艺术形式，区别于其他表演艺术门类的艺术特性和审美原则；
Course Description
进度安排：

第一讲 概念阐释 何谓音乐剧 何谓音乐剧中的五大艺术元素 第二讲 音乐剧的舞台艺术先驱 来自黑人文化的影响 来自世俗文化的影响 来自欧洲文化的影响 第三讲 叙事音乐剧的缘起 富丽秀——美国音乐剧的萌芽 整合性叙事逻辑在《演艺船》中的初步体现 第四讲 音乐剧古典叙事逻辑的确立 叙事音乐剧的序幕 《俄克拉荷马》剧目解析 第五讲 美国叙事音乐剧的黄金时代（上） 《天上人间》中的爱与宽容主题 《南太平洋》中的种族歧视命题 《国王与我》中的东西方文化差异 第六讲 美国叙事音乐剧的黄金时代（下） 《窈窕淑女》中的女性觉醒 《西区故事》中的移民问题 第七讲 摇滚音乐剧的兴起 美国音乐剧与流行文化的分离 《毛发》对于传统的叛逆 第八讲 概念音乐剧（上） 概念音乐剧大师斯蒂芬.桑坦 桑坦音乐剧中的概念化手段 《走入丛林》对于传统童话的解构 第九讲 概念音乐剧（中）《伙伴们》剧目解析 非线性叙事结构 符号化人物设置 抽象化舞台设计 第十讲 概念音乐剧（下）《星期天与乔治同游花园》剧目解析 何谓点描主义 点描主义艺术理念在剧目中的运用 第十一讲 欧洲音乐剧的异军突起 第十二讲 巨型音乐剧的商业逻辑 第十三讲 西方音乐剧现状（上） 何谓百老汇 百老汇的剧院模式 百老汇音乐剧的艺术特征 第十四讲 西方音乐剧现状（中） 百老汇主流音乐剧的分类 百老汇主流音乐剧的审美特征 第十五讲 西方音乐剧现状（下） 何谓外百老汇 实验剧场的艺术逻辑 音乐剧的未来发展 第十六讲 中国音乐剧现状
