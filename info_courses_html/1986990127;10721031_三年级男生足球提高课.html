三年级男生足球提高课

课程内容简介：
秋季学期： 1、技术：巩固、提高基本技术 2、战术： （1）进攻战术：学习和初步掌握2～3人中路进攻配合等 （2）个人防守战术：学习和初步掌握防止对手控球转身的选位和技术等 3、理论知识：越位规则解析等 4、身体素质练习。 春季学期： 1、技术：巩固提高基本技术.学习和初步掌握中路插上射等。 2、战术： （1）进攻战术 ：学习和初步掌握守方罚球区附近任意球进攻战术等。 （2）两人防守配合战术：学习和初步掌握中卫保护边卫等 3、理论知识：假摔的识别和判罚等 4、身体素质练习。
Course Description
进度安排：
按教学周历进行.
考核方式：
秋季学期： 100分。其中4*30米折返跑、踢远、3000米跑成绩各占30%，学习态度10%。 春季学期： 100分。其中4*30米折返跑成绩占20%，踢远成绩占30%、100米跑成绩占20%，引体向上成绩占20%，学习态度占10%。
教材及参考书：
北京体育大学足球教材等。
