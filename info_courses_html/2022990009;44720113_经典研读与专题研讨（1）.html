课程内容简介：
墓志是历史学的重要史料，在中国古代史的学习与研究中地位尤为突出。基于此，本课程将通过通过教师讲授与学生领读的方式集中研读数十方图版清晰、内容丰富的北魏宗室墓志，引导学生在掌握墓志文献的搜集、整理及研究方法的同时，以墓志呈现的历史信息为线索，把握北魏一朝的政治演变、制度沿革、社会变迁等。进而，鼓励学生对墓志中有价值的信息进一步追索，撰写出具有学术意义的历史学论文。
Course Description
进度安排：

第一讲 墓志概论（教师讲授）

第二讲 中古碑志研究的新动向（教师讲授）

第三讲 《元忠墓志》研读（教师领读）

第四讲 教学参观：北京石刻艺术博物馆（或北京房山云居寺）

第五-十六讲 北魏宗室墓志研读（学生领读）

 

 
考核方式：
课堂表现10%：包括出席情况、课堂提问与讨论。 期中报告30%：选取一方北魏宗室墓志进行研读，对其中可能蕴含的研究线索进行阐发。 期末作业60%：在期中报告的基础上进一步充实，完成研究型文章。
