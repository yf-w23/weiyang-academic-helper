课程内容简介：
作为一门思想政治理论课，《中国近现代史纲要》是从历史维度阐述1840年以来的中国巨变及其发展道路，深刻体认中国在现代世界的历史使命，形成认知历史的基本原则与方法。 本课程共计15讲，采取系统讲授的教学形式，辅以课堂互动，避免复杂繁琐的教学形式。 课程考核包括学习参与、课程作业与期末考试3个组成部分。（1）学习参与（10分）：考察学生遵守教学秩序、参与教学活动的情况。（2）课程作业（50分）：考察学生整理史料、剖析事实或历史叙事的能力。（3）期末考试（40分）：开卷考试，独立完成，考试范围是课堂教学内容。
Course Description
进度安排：

本学期的教学安排是每周1讲，共计14讲，均为课堂讲授。

第一讲 作为一种生活智慧的历史及其打开方式（导论）

第二讲 文明是流动的：地理、耕战与中华文明的生长 

第三讲 文明是流动的：科学、竞存与工业文明的跃迁

第四讲 从马戛尔尼使华到鸦片战争：中国危机的实质

第五讲 在工业化与反侵略之间摆荡：早期探索及其失败

第六讲 中国的机遇：十月革命、五四运动与中国共产党成立

第七讲 到达中国的中国共产党：国民革命、苏维埃根据地与长征

第八讲 中国人民抗日战争与中华民族共同体意识的确立

第九讲 中华人民共和国的创立及其制度形态

第十讲 革命红利、苏联经验与中国式现代化的开辟

第十一讲 以苏为鉴、探索失误与“文化大革命”的发生

第十二讲 改革开放、中国特色社会主义与中国式现代化的发展

第十三讲 中华民族现代文明的到来

第十四讲 世界的机遇：中国式现代化道路与人类命运共同体（代结语）
