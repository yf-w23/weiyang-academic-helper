课程内容简介：
本课程将简要介绍热学理论二三十年来的最新进展及其在高新技术（航天航空、微纳电子、激光等）中的重要应用，内容主要是清华大学过增元院士学术团队近几年提出的热学理论体系，在回顾现有的热学及其存在的不足之处的基础上，提出了热的动质二象性说，即热具有能量和质量的双重属性，在热学中引入热质、热质势、热质能（火积）等热学新概念，以建立热量传递的普适定律和优化热量传递过程的普遍原理。通过建立热量传递的动力学控制方程，探讨傅立叶导热定律的物理本质，建立亦适用于极端条件（纳米、超快速、极低温传热等）的普适传热规律；基于变分原理建立传热过程的最小作用量原理，即热质能（火积）耗散极值原理，应用于传递过程和热功转换过程的优化，以提高能量利用效率。本课程在讲授知识的同时，还注重热科学历史、发展现状、研究前沿、以及创新思维和方法。
Course Description
进度安排：

第一讲 热的二象性和热质的概念

第二讲 热质的状态方程及热质的运动

第三讲 热质运动的守恒方程和傅里叶导热定律的物理本质

第四讲 稳态导热定律及其应用

第五讲 热波现象

第六讲 纳米材料的弹道导热

第七讲 芯片热管理科学与技术

第八、九讲 线性非平衡热力学

第十讲 传热过程的热力学优化理论

第十一讲 火积和熵

第十二讲 最小作用量原理和火积耗散极值原理

第十三讲 应用I：导热优化

第十四讲 应用II：对流传热优化
