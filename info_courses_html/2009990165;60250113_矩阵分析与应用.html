课程简介：
矩阵分析理论不仅是物理、力学、土木、电机、航空/航天等传统学科，更是电子、通信、信号与信息处理、系统、控制、生物医疗、量子物理等新兴学科不可或缺的基本数学工具之一。 围绕如何提高工科学生灵活应用矩阵分析这一数学工具来解决实际问题的能力这一宗旨，本课程在内容编排上采用了一种不同于国内外相关教材从数学角度来叙述矩阵理论的全新的体系结构：以矩阵分析的典型应用方法为主线，将矩阵分析理论分为梯度分析、奇异值分析、特征分析、子空间分析和投影分析五大部分。诸如各种广义奇异值分解、总体最小二乘及其推广、特征值分解的推广（矩阵的广义特征值分解、广义Rayleigh商、联合对角化、二次特征值问题、特征分析与Fourier分析）、子空间分析理论及方法、投影分析（正交投影与斜投影）、梯度分析与最优化（尤其是标量函数相对于复向量和复矩阵的梯度计算）、稀疏表示与压缩感知等，是本课程有别于国内外和世界一流大学的矩阵分析或者线性代数课程的特色内容。 除讲授矩阵分析的基本理论外，本课程注重与工程实际的结合与应用。通过多种教学手段，不仅强调矩阵分析基础知识的掌握，更是注重科研思维的培养和科学素养的提升。因此，考核方式不再强调传统的书面考试，而更倾向于考察学生对未知领域的探索能力。
英文课程简介：
This course presents the matrix analysis and applications from the following five parts: gradient analysis and optimizations, the singular value analysis, the eigenvalue analysis, the subspace analysis, the projection analysis. Especially, this course contains the following important applications of matrix analysis: the optimization algorithm designs, the total least squares least methods, the generalized eigenvalue decomposition and the quadratic eigenvalue problems, the oblique projection and so on.
