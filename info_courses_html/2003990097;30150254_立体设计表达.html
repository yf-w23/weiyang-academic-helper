课程内容简介：
本课程旨在通过在一个给定比例的油泥模型设计制作过程中，通过大量的演示和练习，让学生充分理解油泥模型制作的基本规律，训练学生在设计过程中能够合理利用相应材料、工具及手段，快速准确地表达自己的设计概念，取得最佳的展示效果。 教学目标：1、熟练掌握油泥模型制作相关方法和技巧。2、培养对产品的造型要素—体积、质感和复杂曲面的感知能力和表现能力。3、了解各种设计表达手段在模型制作过程中的应用要点。4、提高三度空间及物体形态的创造力及把握能力。
Course Description
进度安排：

1、概论；根据设计命题进行平面设计表达工作包括草图、效果图、关键视图 等

2、依据模型关键视图制作模板、模型底板、骨架、内胎等

3、初敷油泥、粗刮模型找基准面

4、精刮模型1-在模型基本型上进行“势”的推敲

   精刮模型2-按设计方案的平面表达效果图进行细节的推敲

   精刮模型3-明确效果图和模型之间的关系和区别，对模型局部造型进行取舍并统一整体造型风格，

5、模型制作收尾工作，作品展示与答辩，课程总结
考核方式：
考核方式以考查为主: A 胶带图 1套 15分 B 模型模板 1套 10分 C 油泥模型 1个（可分组进行） 60分 D 设计作品的宣讲与答辩 5分 E 课程总结 10分
