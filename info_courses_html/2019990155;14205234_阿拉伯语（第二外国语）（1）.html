课程内容简介：
本课程是零起点学习阿拉伯语的第一阶段。主教材为乔治敦大学阿拉伯语系列教学用书Al-kitaab Arabic language program中的入门教材Alif Baa。辅助教材为外研社出版《新编阿拉伯语》（第一册）。 本课程每次2学时，每周2次，16周学期内课堂用时64课时，分为两个阶段。期中考试前为语音阶段，主要完成字母发音、书写和简单日常会话的学习。期中考试后为初级阶段，主要学习基础语法、课文、情景对话等内容。 课堂时间主要由教师讲解教学重点，进行示范，学生通过大量模仿、练习，旨在从听、说、读、写四个方面掌握阿拉伯语基本技能；我们来自阿拉伯国家的助教会定期到课堂帮助大家纠音和开展对话练习；考虑到阿拉伯语使用地区存在标准语和方言的巨大差异，我们的教学在以标准语为主的前提下，也会在对话环节引入使用广泛的沙姆地区方言，让同学们所学的表达更加地道和生活化。同时，课堂也会结合教学内容向同学们介绍有关阿拉伯伊斯兰世界以及中东地区的历史地理、风土人情等。 在课下，提供相关的课外辅助教材或视听材料，配合一定量的作业以巩固课堂教学成果。阿拉伯语1的学习阶段旨在使学生具备比较扎实的阿拉伯语语音、语法基础，初步了解阿拉伯国家的基本情况，从而产生持续学习的兴趣和动力。
Course Description
This course is the first phase of Arabic learning for beginners. The textbook used for this course is Alif Baa from Al-kitaab Arabic language program of Georgetown University Press
进度安排：

第1-5周 语音阶段，28个字母+第一单元到第三单元的单词和对话。

第6周 期中复习、考试、讲评。

第7-12周 第四、五、六三个单元，核心是单词、语法和对话。

第13-15周 第七、八、九三个单元，核心是单词和对话以及语法的回顾和练习。

第16 周 总复习+口语考试。

 

 
考核方式：
平时成绩（作业和课堂表现） 期中考试（口试和笔试两个部分，口试为线上提交） 期末考试（口试和笔试两个部分，口试为线下随堂）
教材及参考书：
Al-kitaab Arabic language program （Alif Baa） 《新编阿拉伯语》（第一册） 《伊斯兰世界帝国》（商务印书馆）建议作为课外阅读。
