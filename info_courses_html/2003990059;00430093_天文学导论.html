课程内容简介：
本课程是现代天文学的高级入门课程。通过系统学习架构完整的天文学基础知识，学生既能用科学的方法理解宇宙、星系、恒星、行星、生命等的起源、演化与未来，也能深入了解天文学家观测和研究宇宙的基本方法和思维方式。本课程旨在补充和完善本科生在自然科学方面的理论知识体系，启发学生思考人类在宇宙中的存在和地位的意义，建立正确的宇宙观，从天文学的角度提升学生的思维能力。在课程的学习过程中，学生也能了解到天文观测的新发现和宇宙研究的前沿进展。本课程还包括利用校内天文台进行天文观测和在校内开展户外认识灿烂星空等形式多样的实践活动，使学生了解和掌握基本的天文观测方法和技能。实践活动以分组的方式进行，能培养学生的交流、沟通以及合作的习惯和能力。
Course Description
进度安排：

第1章 天体的视运动（星座与星图，天体的周日与周年视运动，天球坐标系，岁差，月相，日月食）

第2章 天体的运动（哥白尼、第谷、开普勒与伽利略的天文贡献，牛顿的万有引力定律，爱因斯坦的狭义及广义相对论）

第3章 辐射与天文望远镜（电磁波，黑体辐射，原子与谱线，多普勒效应，光学天文望远镜，全波段天文望远镜）

第4章 行星系的诞生与演化（太阳系的形成过程，太阳系外行星，地球2.0）

第5章 类地行星与月球（塑造类地行星的过程，类地行星的内部、表面与大气）

第6章 巨行星与卫星（巨行星的特征，巨行星的云、内部、磁场，太阳系的卫星及其地质活动，巨行星的光环）

第7章 矮行星与太阳系小天体（矮行星，小行星，彗星，太阳系残骸，撞击）

第8章 太阳的基本性质（太阳结构，太阳能源，太阳内部，太阳大气与活动）

第9章 恒星的基本性质（恒星物理参数的测量，恒星分类，赫罗图，双星）

第10章 星际介质与恒星形成（星际尘埃，星际气体，分子云，恒星形成过程，原恒星，主序恒星，星团）

第11章 恒星的演化（演化过程，红巨星，行星状星云，新星，变星，超新星与遗迹，元素合成）

第12章 致密星（白矮星，中子星/脉冲星，黑洞，X射线双星，伽马射线暴，快速射电暴）

第13章 星系（星系的分类，银河系，暗物质，类星体与活动星系核，超大质量黑洞）

第14章 膨胀宇宙（星系的发现，宇宙学原理，宇宙的膨胀与哈勃定律，宇宙大爆炸，宇宙微波背景辐射，大爆炸核合成）

第15章 宇宙的大尺度结构（星系团与超星系团，结构的起源，第一缕星光，星系的演化，银河系的形成与演化）

第16章 当代宇宙学（宇宙的物质密度，宇宙的加速膨胀与暗能量，暴胀宇宙，最早期宇宙，多重/平行宇宙）
