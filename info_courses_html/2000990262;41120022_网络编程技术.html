课程内容简介：
本课程将详细讲解基本的TCP、UDP协议的套接字编程，在此基础上进一步介绍高吞吐套接字编程、服务器设计模式、安全编程等内容。课程针对同学作业中常见的设计、实现问题进行讲述，通过结合知名开源项目，介绍中等代码量下实际项目的设计与实现思路。
Course Description
In this course the basic TCP/UDP socket programming will be introduced. More advanced topics like high performance socket programming, design patterns and secure programming will be further illustrated. We will discuss the common design or implementation problems in students' projects, and based on one well-known open source software, we will demonstrate the practical experience in real medium-sized projects.
