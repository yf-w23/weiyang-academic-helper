课程内容简介：
本课程是数据科学入门课程，以Python为核心工具，系统介绍数据科学中的编程基础与典型工作流程。课程首先引导学生搭建编程环境并掌握Python基本语法与数据结构，包括数据类型、逻辑控制与集合操作。随后重点讲解数据组织与输入输出方法，涵盖文件读写与编码处理。在此基础上，课程深入数据操作关键技术，如索引、筛选与多表连接，并介绍文本处理中的字符串操作与正则表达式应用。 课程聚焦实际数据来源与存储，包括通过本地和网络获取数据，以及使用SQL进行数据库查询与管理，同时介绍向量数据库的概念和使用。紧跟技术发展，课程专门设置大语言模型在数据科学中的应用模块，探讨其与数据处理流程的结合方式。数据可视化部分将指导学生创建清晰美观的统计图表。 最后，学生将以小组形式开展期末综合项目，从问题定义、数据获取、清洗分析到结果展示完成全流程实践，巩固所学知识并锻炼项目协作与汇报能力。通过本课程，学生将具备扎实的数据编程基础与解决实际问题的能力。
Course Description
This course is an introductory data science course that uses Python as the core tool to systematically cover the programming fundamentals and typical workflows in data science. The course begins by guiding students to set up a programming environment and master the basic syntax and data structures of Python, including data types, logical control, and collection operations. It then focuses on data organization and input/output methods, covering file operations and encoding processing. Building on this foundation, the course delves into key data manipulation techniques such as indexing, filtering, and multi-table joins, while also introducing string operations and the application of regular expressions in text processing. The curriculum emphasizes practical data sources and storage methods, including acquiring data from local and online sources, as well as using SQL for database querying and management. The concept and usage of vector databases are also introduced. Keeping pace with technological advancements, the course includes a dedicated module on the application of large language models in data science, exploring how they integrate with data processing workflows. In the data visualization section, students will learn to create clear and aesthetically pleasing statistical charts. Finally, students will undertake a comprehensive final group project, going through the entire process from problem definition and data acquisition to cleaning, analysis, and presentation of results. This hands-on experience consolidates their learning and enhances their project collaboration and presentation skills. Upon completion, students will possess a solid foundation in data programming and the ability to solve real-world problems.
