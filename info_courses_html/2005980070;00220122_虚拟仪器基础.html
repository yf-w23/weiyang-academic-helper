课程内容简介：
虚拟仪器是基于计算机的仪器，是利用软件和硬件建立的测试测量系统，用户可以自己定义仪器的功能，实现对各种物理量的测量。目前，计算机测量系统已成为主要的测量工具和研究方向，虚拟仪器作为计算机技术与测试测量技术相结合的一门创新技术，已逐步应用于各行各业中。从前沿的科学研究到广泛的工程应用，再到同学们的科技创新活动，在有需要测量的场合，已少不了虚拟仪器的知识和技术。 《虚拟仪器基础》是一门研究型的设计性实验课程，主要指导学生利用图形化编程语言LabVIEW和相关硬件，构建出个性化的虚拟仪器，从而提高学生的创新和动手实践能力。该课程为学生提供了一个理论与实践相结合、融合运用所学的多门课程知识的机会，为将来从事有关测量方面的科研任务或科创竞赛，尤其是构建计算机化的测控仪器及系统打下基础。 该课程面向全校理工科各专业的本科生,在虚拟仪器实验室进行，每个实验组都配备有计算机（安装有LabVIEW）、NI公司的数据采集卡、ELVIS III、MyDAQ、MyRIO及必要的外部设备。MyDAQ和MyRIO在选课期间学生可以借回去在课下使用，方便同学们随时随地的动手实践。
Course Description
进度安排：

1）前8周采用边讲边做的模式，讲授虚拟仪器设计所需的基本知识。

先讲授某个知识点，然后指导学生利用该知识点解决某个问题，即学即用，帮助学生快速掌握LabVIEW程序设计的方法和相关硬件的使用。

2）后8周采用基于项目式的学习方式（PBL），指导学生完成一个虚拟仪器设计大作业。

包括选题、查阅文献、讨论、实践和总结等环节，利用LabVIEW和相关硬件，构建出个性化的虚拟仪器，完成一个具有实际应用背景的项目。该项目选题可以在教师提供的选题范围内根据自己的兴趣选定，也可以自拟题目并征得教师同意。所选择题目难度适中，鼓励创新和合作。在课程设计的整个过程中，都将得到授课教师的指导。
