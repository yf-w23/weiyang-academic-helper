### 课程内容简介

自19世纪末期萌发的现代主义运动是世界建筑史上最为剧烈的变革之一。无论是建筑形态、建造方式还是理论论述，现代主义运动以及之后的建筑发展均与之前相对连续的建筑传统形成了强烈的反差。因此，“外国近现代建筑史纲”课程将讨论19世纪以来现代主义运动孕育、成熟、衰落以及此后外国建筑发展的历史。该课程不仅帮助学生理解近一百多年来激烈建筑变革的根源与过程，同时也为他们当下的设计训练提供思想养分。在综合吸收最新现代建筑史研究成果的基础之上，本课程将侧重讨论如下几方面内容：导致现代建筑出现的危机与条件；现代主义运动早期的实验性探索；代表性建筑师的思想与作品；现代城市规划理论的形成；现代主义盛期建筑与城市的面貌；对现代主义运动的批判与反思；以及20世纪末期以来新的理论与实践。教学目标是阐明历史发展脉络，厘清主要理论潮流实践倾向，并且对不同立场的优缺点进行批判衡量。 在获得基本历史素养的基础之上，建筑学本科生还将通过此课程了解主要的现代建筑与城市理论，学习相应的设计策略，初步确立对现代建筑发展的观点以及自身的建筑立场。

## 晚清文学与文化

| 字段 | 内容 |
|------|------|
| 课程编号 | PK02035000 |
| 课程名 | 晚清文学与文化 |
| 总学时 | 34 |
| 总学分 | 2 |
| 开课教师编号 | L254PKJS |
| 开课教师 | 北大教师 |

### 课程内容简介

本课程将在文化史的视野下，概说晚清各种文学现象及思潮，按照较为广义的“文学”范围，重访这一时期印刷、出版、阅读、书写文化的变革，呈现诗、文、小说、图像等各体文学发展的内在肌理。与此同时，课程更将尝试探索文学为代表的“读写文化”与近代中国政教制度、社会结构、文化空间等外部环境之间的互动关系。
本课程所涉的“晚清”，是指1860-1911五十年间，亦即从“同治中兴”到新文化运动，传统中国的政教体系受到外力冲击，开始艰难转型为近代国族的时期。梁启超曾称之为“过渡时代”。然而，“晚清文学”的意义，绝不止于充当古代文学到现代文学的过渡。这一时期的文学与文化，既是中国古典诗文小说戏曲、文体学、诗文评传统集成的时代，又是这些文学传统接受外来影响，激发内在活力，引发自身反思的阶段，更是近代意义上文学观念、文学学科、文学教育奠定的时期。从文化史的角度看，晚清文学既不是古典文学的黯淡结尾，也不仅仅是现代文学的早期萌芽。晚清文学自有其性格，它孕育了文白雅俗并陈的多元文体格局。古今中西各种读写体验并存，使其成为中国文学史上一个特殊的时代。
本课程的研讨对象，既有曾国藩、梁启超、章太炎、刘师培等对于“文学”言说和书写经验产生重大影响的人物，也有桐城—湘乡派、同光体、新文体等文学史上的重要体式流派。此外，伴随着出版网络、报刊媒介、都市空间和新教育空间的建立，白话文体、翻译文学、图像文本、旅行文本等文学现象，也在考察范围之内。

## 微纳米测量与测试技术

| 字段 | 内容 |
|------|------|

| 总学分 | 3 |

| 开课教师 | 赵晓光 |

### 课程内容简介

《微纳米测量与测试技术》该课程的开设，是基于当前微米纳米尺度的微观领域科技研究的迫切需求，是“设计-制造(工艺)-测试”这一全过程中必不可少的核心组成内容。
    (1)本课程主要针对微纳米技术、微机电系统(MEMS)研究中的关键测量和测试问题，结合学术前沿和国内项目需要，讲授针对微纳米尺度的结构和工艺的各种测量方法。主要涵盖5个方面内容：光学测试技术、非光学测试技术、微加工测试技术、微结构的性能测试以及产品封装后的二次电性能测试。对于微纳结构和微纳器件，测试覆盖整个产品的过程。每部分内容的讲述都是从测试目的、测试方法的原理，测试设备的目前发展情况和使用情况，并结合测试设备和相关的测试结果来深入浅出的讲述不同测试方法所适用的测试范围，同时给出测试方法所具有的优点和不足。
    (2)课程结合一些在测试领域最新的研究成果，讲述新的微纳结构的测试方法，并让每位同学针对自己的课题等讲述自己设计的或者别人公开文献上发表的先进的微米纳米测试方法。
    (3)针对课程上讲述的相关内容，安排学生的半观摩、动手性实验，在辅助老师的指导下完成DIY AFM、激光多普勒、白光、STM电镜、全息等现代化的测试实验内容，进一步了解实验，掌握仪器，理解微纳测试方法。

《超精密光栅尺无限速高细分信号处理技术研究》作为《微纳制造与IC装备（1）》课程的重要专题实践，聚焦于超精密光栅位移测量领域中的关键技术挑战。学生将在教师指导下，深入研究高速测量中细分率受限与低速测量中高阶周期非线性误差等实际问题，系统掌握从误差建模、算法设计、MATLAB仿真到FPGA硬件实现的完整工程路径。实践内容涵盖相位恢复算法开发、高阶误差抑制方法设计、软硬件协同优化等，旨在提升学生面向实际工程的分析、解决问题能力。

### 课程内容简介

微生物学基础实验主要内容包括微生物个体和菌落形态观察，用显微镜对酵母培养液进行酵母大小和含量测定，普通染色和革兰氏染色，培养基制备和灭菌，无菌操作，周围环境中微生物检测和分离,菌种鉴定，生理生化反应，影响微生物生长的因素和细菌生长曲线测定和培养条件的优化，紫外诱变育种等内容。整个实验课中贯穿了显微镜油镜的使用，无菌操作和发酵应用，以及对环境中分离的一株未知菌进行鉴定过程，微生物学需掌握的基本操作在实验中得到反复训练，同时加深了同学们对微生物学的基本概念的理解和掌握。实验设计贴近生活,大大加强了同学们对生物学学习的兴趣。本课程特别强调光学显微镜油镜技术和无菌操作等微生物基础技术掌握，而掌握微生物无菌操作技术是选修细胞生物学实验,遗传学实验和分子生物学实验的基础课程,在实验报告中注意要求规范报告书写和科学数据处理，为同学们将来参加生命科学研究和医学临床打下坚实的基础。微生物学基础实验通过显微镜下的微生物观察及染色实验，酵母大小和含量测定、微生物的分离纯化与鉴定、微生物的生长及代谢研究、设计型课题实验方案设计四个板块将微生物学基础实验技术串连起来，在加深对微生物学的基本知识的理解和掌握的基础上，给同学提供一个课题设计活动的体验。实验设计围绕经典微生物模式菌株和课堂上分离的未知菌两条线展开，既包含基础验证性实验，也有自主综合型课题设计干实验。设计性实验占课程内容的20%。课堂引进学科先进实验设备全自动微生物微液滴培养系统MMC应用于教学实验，有效缩短原有实验时长的基础上，可以开展更为深入的生长特性研究。课程强调培养学生观察、分析和解决问题的能力，建立科学研究的思维和方法。通过7周小综合基础实验和期末实际操作考试实验，使课程安排紧凑，期末成绩报告约占2/3,实际实验操作考试成绩约占1/3，使实验课程成绩评价更加合理。本课程适合生命学院、致理书院、医学院、药学院和化生大类同学，想在生命科学方面发展的大一下，大二，大三的同学选修。本课程48学时2学分。

### 课程内容简介

本课程旨在引导学生深入理解文化创意设计的核心理念，从文化根源出发，通过设计语言表达文化内涵，创造出具有市场竞争力的文化创意产品。课程主要分为文创的概述、文创设计的精神内核、文创设计的构成要素、案例赏析与设计实践四个部分。首先解析文创概念、发展历程、分类特征及社会价值，为后续学习奠定基石。随后，深入挖掘东西方传统文化精髓，对比分析不同文化视角下的设计哲学，将传统元素与当代语境相融合，强调文化与创新思维在现代设计中的应用。
文创设计课程是专业能力的综合展现。课程重心聚焦文字、图形、色彩与媒介等文创构成要素，通过授课与实践相结合的方式，培养学生的设计感知与应用能力，提升学生审美水平。

### 课程内容简介

物理有机化学主要是由有机化合物的结构研究和反应历程的两方面来探讨有机化学。
它的理论基础主要是量子化学和以此为依据的化学键理论和电子理论。关于有机化合物的结构的研究方法以物理学研究为主，化学方法只起辅助作用。例如把一个有机分子中的取代基对该化合物的化学性能的影响归结为两种电子效应，即诱导效应和共轭效应，其效应的强弱即表明某一取代基吸引电子或排斥电子能力的大小，这一概念就是根据上一世纪20年代将物理学中的电子理论引入有机化学中。
反应机理是研究反应发生的实际过程,包括反应进行的方向,途径及其规律，研究反应机理可以帮助我们把许多看起来无关的反应联系在一起,找出它们的共同规律，用以指导反应条件的选择，达到提高反应选择性和产率的目的,并可作为新的合成反应的依据。反应机理研究是基于有机化合物中电子的分布或势能面。
另外，物理有机化学在目前已扩展并包含了其他学科领域，如生物有机化学，金属有机化学，材料科学和超分子化学等，这些新的学科给予物理有机化学研究结构和反应性相互关系提供了肥沃的土壤。
