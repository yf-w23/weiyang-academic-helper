### 课程内容简介

课程计划以环境科学、气象学、气候学、大气科学、地球科学相关课程的科学基础为素材，通过课程讲授、课外阅读、实验实践、课堂展示和小班研讨等环节，使学生深入理解气候变化原理及相关问题。

## 气象学基础

| 字段 | 内容 |
|------|------|
| 课程编号 | PK12733060 |
| 课程名 | 气象学基础 |
| 总学时 | 34 |
| 总学分 | 2 |
| 开课教师编号 | L254PKJS |
| 开课教师 | 北大教师 |

### 课程内容简介

气象学是研究大气现象和各类天气事件的科学，和海洋学、地质学、天文学等同属于地球科学的范畴，融合了物理、化学以及流体力学。天气现象及变化，特别是一些极端天气，例如强降水、高温和干旱，深刻影响社会经济运行和人类日常生活。因此气象学的研究和应用对人类生存和发展具有重要意义。为此，我们需要认知不同天气现象背后的物理原理，掌握不同尺度天气过程的基本规律，从而更好地预测预报天气。

在全球范围内，发掘不同环境问题的产生机制和应对策略也有赖于我们对气象学的认识。其中涉及空气污染、极端野火、臭氧层空洞等严峻的环境挑战，而这些与不同尺度的天气和气候过程息息相关。另一方面，近些年来新能源产业的发展，特别是风、光、水资源，需要气象学理论和应用为新能源的合理建设和运行提供支撑保障。将经典气象学理论与这些新问题和新需求结合，成为目前该学科发展的重要方向，衍生出了污染气象学、风能气象学、农业气象学等分支学科。

本课程以各类天气要素和天气现象为切入点，阐释其中的基本物理原理；初步介绍定量分析大气现象和天气过程的方法；着重从环境科学与工程的视野，介绍气象学在不同环境学科方向的交叉和应用。此外，本课程将介绍人工智能在气象学领域的发展趋势和应用前景。

### 课程内容简介

新能源汽车技术革命支撑了中国汽车产业由大到强的弯道超车过程，发展出了以新能源动力系统为代表的创新技术体系，包含了内燃机、驱动电机、动力电池、燃料电池等创新技术。
汽车构造（1）是车辆工程专业的基础核心课程，是车辆工程专业学生走向专业学习的入门课。课程旨在培养和提高学生对于汽车动力系统的直观认知能力，掌握汽车动力系统的典型结构，提升实际动手操作能力，为车辆工程专业其它课程的学习及从事汽车设计和研究等工作打下坚实的基础。
通过本课程的学习，学生将建立对汽车功能实现的形象认知，搭建汽车原理与现实结构之间的物理桥梁，使得学生全面认识汽车动力系统构型及其关键动力装置，掌握内燃发动机、锂离子电池、燃料电池、驱动电机的结构功能与基本原理。
本门课程的知识面广，学科跨度大，课程会采用对比的方式帮助大家快速理解汽车动力技术的发展历程，并帮助大家初步构建面向总体的思维能力。
这门课程也是低年级同学熟悉车辆学科的重要课程，对于前序课程没有具体要求，重点在于课堂学习与知识体系理解，可以为后续相关汽车理论学习和具体专业问题的深入研究提供工程基础知识。

| 总学分 | 3 |

| 开课教师 | 李克强 |

### 课程内容简介

本课程以力学为基础，阐述汽车的行驶性能(动力性、燃油经济性、制动性、操纵稳定性、行驶平顺性以及通过性），研究在满足这些性能要求基础上选择汽车设计参数的相关原则，并介绍如何利用现代电子及控制技术提高汽车行驶性能。
    本课程分析了汽车纵向、横向和垂直方向的汽车动力学模型，介绍了汽车行驶方程式、汽车运行油耗方程式、二自由度汽车操纵稳定性和平顺性模型，在此基础上对汽车使用的各种基本性能(包括动力性、经济性、制动性、操纵稳定性、平顺性)的基本概念和评价方法进行了讨论。不仅介绍了基本性能相关的国家标准和发展方向，也介绍了汽车性能的具体计算方法，还要求学生编程进行计算。同时还分析了汽车的基本性能指标和汽车结构参数的关系。课程注意将新的技术引进教学，讨论了混合动力汽车、ABS和ESP的基本概念和原理。课程对汽车的越野性基本不讲。

## 汽车探秘——奇妙的构造

| 字段 | 内容 |
|------|------|
| 课程编号 | 00150252 |
| 课程名 | 汽车探秘——奇妙的构造 |
| 总学时 | 32 |
| 总学分 | 2 |
| 开课教师编号 | 2011990176 |
| 开课教师 | 刘亚辉 |

### 课程内容简介

1、	汽车底盘和传动系概述：讲解汽车的发展简史、汽车底盘各组成部分的作用及发展趋势；重点讲解各种动力传动总成装置布置形式的特点。
2、	离合器和手动变速器：介绍离合器和手动变速器的发展简史，结合实物拆解分析来讲解离合器及变速器的作用、种类、结构和工作原理；重点讲解膜片弹簧的弹性特性及齿轮传动的工作原理及发展过程。
3、	自动变速器：介绍自动变速器种类和发展简史，结合实物讲解液力变矩器的工作原理和行星齿轮变速器结构原理及特点，介绍双离合变速器和机械无级变速器的结构和工作原理。
4、	混合动力电动汽车变速器：介绍混合动力电动汽车的发展简史，结合实物讲解混合动力系统的类型、P2混合动力变速器的结构和工作原理、串并联式混合动力变速器的结构和工作原理、功率分流式混合动力变速器的结构和工作原理、混合动力变速器的润滑和冷却、插电式混合动力电动汽车的工作原理。
5、	最终传动：结合实物讲解万向传动装置、主减速器、差速器结构和工作原理，重点讲解锥齿轮传动的特点及差速器工作原理。
6、	四轮驱动；讲解四轮驱动的类型、结构和工作原理。
7、	行驶系：结合实物认知行驶系主要部件：车架、悬架、车桥和车轮；重点讲解子午线轮胎的优点、各类独立悬架的结构及特点；一般讲解车轮定位，主动悬架工作原理。
8、	转向系：结合实物讲解转向操纵机构、转向器、转向传动机构的结构和工作原理；讲解液压助力转向和电动助力转向系统的结构和工作原理，介绍线控转向系统新技术。
9、	制动系；结合实物讲解制动器类型、结构和工作原理；讲解伺服液压制动系统、驻车制动系统和气压制动系统的结构和工作原理，介绍汽车防滑控制和主动制动系统等新技术。
10、2次周末实践环节：北京汽车博物馆参观讲解，以及一次北京市高级别自动驾驶示范区参观体验。

### 课程内容简介

微生物广泛分布于土壤、空气、水等多种生境中，微生物学是研究各类微小生物生命活动规
律和生物学特性的科学，既是现代分子生物学、遗传学等学科的基础，也是一门应用性、实
践性很强的前沿学科，在食品工业、生物制药及环境保护等诸多领域都发挥着重要作用。
微生物学主要经历了三个时期：（1）形态学时期。继发现微生物世界以后的200年间，微生
物学的研究主要聚焦于形态描述和分门别类阶段。（2）生理学时期。19世纪中期，以法国
的巴斯德和德国的柯赫为代表的科学家将微生物的研究从形态描述推进到生理学研究阶段，
建立了分离、培养、接种和灭菌等一系列独特的微生物技术。（3）现代微生物学时期。微
生物学发展到从生化水平及分子水平进行微生物的研究，同时推动了生命科学各领域的飞速
发展，微生物学与人类的生产生活也越来越紧密。
《趣味微生物实验》作为一门全校通识实验课，旨在让非生命科学专业的学生通过开展有
趣、易行、紧跟热点且与生活关系密切的微生物实验来更加直观地了解微生物学的学科发
展，激发学生对生物学的兴趣，培养探索精神，健全育人培养体系，进一步深化人才培养模
式。
本课程遵循“微生物学科发展的三个主要时期”这一逻辑主线，设置一次先导课及四次实验
课，包含了微生物培养、显微镜使用、微生物染色、微生物分离、无菌操作、产物检测等微
生物技术。
先导为生活中的微生物专题讲座及微生物实验的安全培训。
实验一为周围环境中微生物的分离和观察，用平皿培养和稀释分离法检测环境及体表的微生物，认识微生物的基本特征、微生物存在的普遍性及与人类共存的方式。
实验二为工程化大肠杆菌与野生型菌的染色与观察。通过染色并在镜下观察经诱导改造的工程菌与野生型大肠杆菌的形态学差异及染色特性，让学生了解微生物学研究在合成生物学发展中的应用和意义，体会科学发展给社会进步带来的益处。
实验三为口腔微生物的生物成膜及活死菌染色观察。让学生了解口腔这个特殊环境中微生物对健康的影响。
实验四为口腔微生物群的质谱检测。学生通过该实验不但可以更为全面地认识口腔中的微生物群体状态，更可以学习不同的微生物鉴别的方法。

### 课程内容简介

全球变化水文学是地球系统科学专业的核心基础课程，与气候学、生态学、水文水资源、水力学等课程联系紧密。本课程聚焦全球变化水文学的基础知识与核心理论，着重培养学生定量分析地球水循环变化规律的能力，以及探究这些变化背后自然与人为驱动因素的方法。课程整合了多尺度水文观测与数值模拟领域的前沿科研成果及学术见解，涵盖原位观测网络、卫星遥感、陆面水文建模等方面的新技术与新方法。以水循环为核心纽带，课程系统阐释人类世复杂地球系统演变的驱动机制，使学生全面掌握：水圈变化的基本理论及分析计算方法；全球水系统观测、模拟与未来演变趋势预估方法；水循环变化对社会经济发展和人类福祉的影响机制及风险评估方法；以及应对全球水循环失衡的韧性提升理论与实践路径。

## 全球变化与可持续发展

| 字段 | 内容 |
|------|------|
| 课程编号 | 00460063 |
| 课程名 | 全球变化与可持续发展 |

| 开课教师 | 白玉琪 |

### 课程内容简介

课程主要讲授和研讨地球系统的结构和功能，主要议题包括全球尺度的生命和生物多样性、生态环境、水资源、农业和土地利用、人口、城市、能源、气候、经济、卫生、自然灾害的原因和影响。课程讨论当前和未来全球环境变化、特别是气候变化下已经或者可能采取的应对策略。课程鼓励学生从多学科视角就上述问题进行深入研讨。课程培养对学生对于全球变化问题的独立研究能力。


讲座课主题：
-地球环境演变
-全球生态
-全球人口与发展
-全球水资源
-全球粮食与农业系统
-全球土地资源
-全球能源转型与可持续能源
-工业化、经济增长与可持续发展
-全球气候变化
-全球自然灾害
-全球公共健康与生态安全
-全球治理与国际合作
-青年行动与个人/社会选择
-中国传统生态观与可持续发展

### 考核方式

课程采用考察方式进行考核。考虑学生如下四个方面的表现：

1) 讲座课出勤和随堂考查

2) 每次书面作业表现（12次课）

3) 小组作业质量和表现

4) 个人课程大作业
