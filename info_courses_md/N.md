## 脑科学导论

| 字段 | 内容 |
|------|------|
| 课程编号 | 20950014 |
| 课程名 | 脑科学导论 |
| 总学时 | 64 |
| 总学分 | 4 |
| 开课教师编号 | 2020990093 |
| 开课教师 | 刘嘉 |

### 课程内容简介

本课程将系统地带领学生探索从单个神经元到复杂脑网络的组织与功能层次。课程分为三个模块，每个模块分别从不同的角度深入探讨大脑的结构与功能，结合先进的神经科学技术，帮助学生理解神经元、神经核团以及整个脑网络如何协同工作，形成复杂的认知和行为功能。（1）神经元。这一模块将聚焦于神经系统的基本功能单元——神经元。这一模块将介绍神经元的基本属性，如动作电位的产生、突触传递、以及神经元的兴奋性与抑制性网络构建。学生将通过实例了解神经元之间如何通过复杂的连接形成微观网络，进而推动信息的传递与处理。课程内容将基于小鼠和猴子的大脑研究，特别关注钙成像和光遗传学技术的应用。（2）核团和脑功能区。这一模块将主要探讨神经元群体在认知功能中的角色。核团和脑功能区是大脑中的中间结构，具有特定的解剖结构和功能。本模块将关注它们在感知、决策、运动控制和认知功能中的具体作用。课程内容讲重点集中于非人灵长类的大脑研究，学生将学习如何通过超声成像和神经电生理技术探测这些脑区的功能。（3）脑神经网络。这一模块将聚焦在于大脑不同功能区构成的宏观网络，内容将涵盖脑区间的结构和功能连接、脑网络组织模式，以及在不同的感知和认知任务中如何实现信息交互和功能整合。学生将学习理解如何通过磁共振成像技术研究人类大脑的不同区域的结构和功能，以及在认知、情绪和社会行为以及脑疾病中的作用，学习脑网络的功能分离与功能整合。

### 课程内容简介

本课程针对测量的基本概念和内涵，以及能源动力系统中重要参数的测量方法与仪器，将包含课堂讲授和实验实践环节，二者交叉同步进行，内容相互对应。
1）	课堂讲授部分主要包括三大部分：第一部分为测量理论基础，讲授测量相关的基本概念、测量误差与标定、测量量系统分析、仪器种类、性能特征等知识，并结合实践环节讲授数据采集和信号处理基础与方法、LabVIEW软件编程、光学系统设计等；第二部分主要针对能源动力系统所涉及的关键测量目标，具体讲授不同物理量的测量定义、原理及方法，主要包括温度测量与控制、压强测量、流动速度与流量测量、气体密度与浓度测量、位置与距离测量（遥测）等；第三部分为拓展性内容，主要包括对现代传感器及智能设备的介绍等；

### 课程内容简介

《能源化学基础》是面向自强书院同学开设的普化类初级课程。本课程旨在系统讲授能源物质与物质结构基础；化学热力学、动力学与能量转化；氧化还原反应与电化学能源等核心基础理论，并阐释其在能源获取、存储、转换与传输过程中的应用。
讲授内容主要包括：
1. 能源物质与物质结构基础：从能源利用与能量转化的视角，系统讲解原子/分子结构、化学键、晶体结构及物质状态等核心知识，重点阐明不同类型能源物质（如化石燃料、电池电极材料）的“结构-性质”关联，解释其能源应用特性的差异。
2.化学热力学与能量转化方向及限度：讲解化学热力学基本原理（热力学第一/二定律、热力学能、功、热、焓、熵、吉布斯自由能等），帮助学生深入理解能量的本质属性及转化过程的方向与限度规律。
3.化学动力学与能量转化速率：系统讲解化学反应速率、活化能等基本概念，分析浓度、温度等对化学反应速率的影响，引导学生掌握能量转化的速率规律与优化途径，同步介绍能源催化等学科前沿。
4.氧化还原反应与电化学能源：系统讲解氧化还原反应、化学势与电化学势、电极电势、能斯特方程、原电池等基础理论，结合锂离子电池、燃料电池等原理及构造的介绍，理解电化学能源存储与转换的核心机制，建立理论与技术的关联。
5.能源化学的现代应用：紧密衔接前述基础理论，串联讲解燃烧化学、二氧化碳化学转化、电化学能源技术（燃料电池、锂离子电池、电解水制氢）、光能源化学与光伏电池等现代应用，并通过 “能源应用课程报告” 的实践形式，引导学生主动调研、分析能源领域实际问题，强化理论应用能力。

### 课程内容简介

本课程作为自强书院招生后一年级本科生的专业基础课，主要介绍跟能源有关的经济学基本理论、各类能源系统和市场的基本知识、能源经济学在中国国情下实践的具体案例等。授课主要采用课堂讲授及讨论的形式，并通过定期作业、课堂表现、期末考试（第16周）共同进行成绩评定。
课堂主要讲授：（1）能源经济绪论，包括能源经济学的定义、角色与基本概念等；（2）能源的需求与供给，包括价格弹性与收入弹性、非完全竞争市场、工业能源需求与石油市场等；（3）煤炭市场与天然气市场，包括非可再生能源资源的最优开采、多元回归分析、煤炭垄断与市场力等；（4）电力市场，包括市场结构与竞争机制、电力商品特殊性、我国电力改革历程等；（5）各类发电资源的经济特性，包括成本体系，火电、核电、水电、风电、光伏、储能的经济性分析等；（6）市场组织与定价，包括价格、效率与公平，电力市场中的价格体系等；（7）电力市场监管，包括市场失灵与政府监管、电力市场监管案例等；（8）能源金融衍生品，包括风险偏好、风险分担与对冲，远期合约与期货等；（9）能源效率与外部性，包括能源生产率与效率、外部性、能源模型工具等；（10）气候政策与碳交易等，包括碳定价与碳排放市场、科斯定理与排放权交易等。
通过本门课程，希望学生能在修习《能源导论》后获得能源系统的物理认知之上，进一步建立能源系统的经济认知体系，实现对于能源系统更加全面丰富的认知。课程将支持学生掌握经济学中跟能源有关的基本理论，具备使用基本经济学工具进行计算分析的能力，较为深入地理解建立能源市场体系的背景与意义，并充分认知我国能源市场与经济体系的特殊性，学会进行能源领域的中外对比经济分析。

### 课程内容简介

鸟类是脊索动物门脊椎动物亚门鸟纲的所有动物的统称，它们体表被覆羽毛、有翼、恒温、卵生、新陈代谢旺盛，常具飞行能力。作为日常生活中的常见野生动物，鸟类往往具有靓丽的羽色或者婉转的鸣唱，深受人们的喜爱。除此之外，作为生态系统的重要组成部分，鸟类在维持生态系统稳定、防治虫害与鼠害、为人类提供肉与蛋及装饰品、作为仿生学与流行病学研究的对象等方面发挥着无可替代的作用，存在巨大的研究与保护价值。在全球变暖、极端天气事件频发的当今，许多鸟类的生存状况正在各种因素的综合作用下受到严重威胁。增进对鸟类的了解，将有助于我们保护这一动物类群，进而扭转全球生物多样性下降的趋势，推进我国乃至全世界的生态文明建设，促进人与自然和谐共生。本课程将对鸟类的起源与演化、多样性与分类、形态与生理、羽毛与鸣声、繁殖与迁徙、生态与保护等进行介绍，并通过观鸟、样线调查、鸟撞调查等实践活动丰富授课形式，以期增进各专业学生对鸟类的了解。
