## 迈向通用的人工智能

| 字段 | 内容 |
|------|------|
| 课程编号 | 00240401 |
| 课程名 | 迈向通用的人工智能 |
| 总学时 | 16 |
| 总学分 | 1 |
| 开课教师编号 | 2013990148 |
| 开课教师 | 刘知远 |

### 课程内容简介

近年来人工智能正以前所未有的速度改变着我们的世界。为了帮助同学们快速拥抱时代变革，本课程将介绍人工智能，尤其是大语言模型技术的最新进展，帮助学生理解其发展历程、关键技术和面临的挑战。课程还将和同学们一同展望迈向通用人工智能的技术趋势，并讨论智能技术给我们社会带来的机遇与挑战。

本课程内容安排由潜入深，包含有大模型的基础、前沿方向与应用。首先，课程将讲述通用人工智能概述，回顾人工智能的起源、发展历程和当前的技术水平，帮助学生建立对人工智能的宏观理解。接下来，课程将深入讲解大模型的基础知识，包括大模型的训练方法与网络架构，帮助学生掌握大语言模型背后的运作原理。随后，课程将进一步介绍大模型的前沿领域，如多模态智能和自主智能体，探索如何通过融合多感官输入以及复杂行为决策，来构建先进的智能系统。在此基础上，课程还将探讨人工智能的交叉应用，展示人工智能与其他领域的结合，并讨论人工智能的安全与伦理问题，引导学生思考智能技术应用中可能带来的风险和挑战。这一部分旨在让学生认识到，随着人工智能技术的广泛应用，理解其潜在影响和道德约束变得尤为重要。

本课程将贯彻"用AI传授AI"的创新理念。课前由AI进行内容讲解，课上则由教师回顾课程内容并引导同学们深入探讨前沿开放性问题。这种教学模式不仅让学生在学习AI的过程中亲身体验和使用AI，还使他们置身于教育智能化变革的前沿，切实感受大模型技术的发展现状及其对日常生活的深远影响。同时，“用AI传授AI”的一大优势在于AI授课时能与每位学生实时互动，从而实现真正的个性化教学。AI的参与也使得人类教师能够将更多精力聚焦于前沿开放性问题的讨论，培养学生的批判性思维能力。

滤波是一种从看似杂乱无章的测量数据中提取能够为我们服务的信息的方法。其应用涉及到自动控制系统设计、通信信号处理、图像处理、地质勘探、经济预测等多个工程领域。本课程结合具体工程事例，介绍滤波理论在实际应用问题的刺激下，从原始的最小二乘估计发展到维纳滤波器、卡尔曼滤波器、鲁棒滤波器、粒子滤波器的过程；着重讨论如何从工程要求出发凝炼理论问题并以恰当的数学工具加以解决；充分体现工程技术中蕴含大量实际的数学问题，而数学的进步又为技术发展铺平道路的绚丽景象。本课程还将初步探讨最小二乘与大数据分析、机器学习、人工智能等的密切关系。
