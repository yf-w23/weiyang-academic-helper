## 劳动耕读实践

| 字段 | 内容 |
|------|------|
| 课程编号 | 14700312 |
| 课程名 | 劳动耕读实践 |
| 总学时 | 64 |
| 总学分 | 2 |
| 开课教师编号 | 2009990137 |
| 开课教师 | 梅赐琪 |

### 课程内容简介

“劳动耕读”实践是新雅书院自2019年探索出的具有书院特色的劳动教育形式。2022年，新
雅书院在海淀郊区建设的“新雅农场”正式投入使用。本课程主要是利用新雅农场的实践基
地，在学期中开展一边劳动（“耕”）一边读书/学习（“读”）的劳动教育形式，把智育
和劳育充分结合起来。为从小很少有机会接触生产性劳动的同学提供劳动实践的机会。本课
程将安排学生从春季春分开始，利用周末的时间，在专业技术人员的指导下，按照24节气的
规律，选课同学个人或分小组在新雅农场进行承包种植，从翻土、播种、施肥、浇水、管理
和收获均全过程负责，种植北方的农作物。最后课程的考核依据种植效果或收获、学生撰写
的劳动心得以及日常的投入程度。
本门课程的目标是：一是落实国家关于大学生劳动教育的要求，结合古代中国教育的耕读传
统，探索“五育并举”的劳育和智育的结合；二是通过生产劳动锤炼学生的心性，使学生亲
近大地，了解国情，直观感受劳动的不易，培养学生健康的劳动观和价值观，培养跟广大劳
动人民的共情能力，实现立德树人的目标。三是通过体力劳动和脑力劳动结合，调节学生在
学校繁忙的学习节奏，促进学生身心健康。

## 量子力学研讨课

| 字段 | 内容 |
|------|------|
| 课程编号 | 30430272 |
| 课程名 | 量子力学研讨课 |

### 课程内容简介

量子力学是物理系学生最重要的一门专业基础课，也是学生以后继续从事物理学相关科学研究的最重要的基础知识。
从事当今物理学主要的研究领域，如凝聚态物理，原子分子与光物理，粒子物理与核物理，乃至天体物理和宇宙学，
无不用到量子理论。物理系本科生对于宏观世界的描述已经具有了很好的概念和数学上的基础，而且在现实世界中有
很多看得见摸得着的东西可以让学生相信这些描述是正确的。但是在学习量子力学之前，他们对于微观世界的接触很
少，而且量子力学对微观世界的描述与牛顿力学对于宏观世界的描述看起来截然不同。所以量子力学被认为是物理系
本科阶段最难修的一门课程。学习量子力学的难点并不在于其数学，而在于如何理解量子力学对原子分子层次的微观
世界的描述，在于如何发自内心的相信量子力学是描述微观世界的正确理论。一般来说，对于量子力学的学习，必然
会经历三个阶段：首先在刚开始学习的时候，看到量子力学与已有的宏观经验格格不入，不禁会问：量子力学是正确
的吗？学习了一段时间，接触了量子力学的理论框架和一些实验验证以后，会逐渐认为量子力学应该是正确的。再往
后会发现量子力学的基本概念中会有很多无法理解的东西，又不禁要问：量子力学真的是对的吗？需要指出的是，虽
然量子力学的应用取得了巨大的成功，但是对于量子力学基本概念的理解（例如测量和诠释），仍然是还没有彻
底解决的前沿问题。

本课程作为量子力学研讨课，与《量子力学》课程同步。教师首先提出问题，学生预习、调研和思考，并提出更多的
问题。在课堂上学生通过报告的方式讲述自己对于相关问题的理解，并与教师和学生就不同的观点展开讨论或辩论。
在激烈的讨论中，教师努力引导学生正确理解量子力学中的疑难问题。同时当然也会产生更多的可能在课堂上也无法
立刻解决的问题，学有余力的学生可以进一步进行研讨。

### 课程内容简介

《Renal urology》是一门学分为4的课程，旨在通过三到四周的学习，使国际医学生掌握泌尿系统的解剖
学、生理学、疾病诊断和治疗等基础知识，了解常见的泌尿外科手术技术，并培养其在泌尿外科领域的临床能力
和团队合作意识。
课程涵盖了以下内容：

1. 泌尿系统解剖学与生理学基础：包括泌尿系统的结构与功能、肾脏生理学、肾上腺生理学及解剖和膀胱解剖
与生理等。

2. 泌尿系统常见疾病与诊断：包括泌尿系统感染的诊断与治疗、泌尿系统结石的形成机制与治疗策略、泌尿系
统肿瘤的癌症类型、分期及治疗选择等。

3. 泌尿外科检查与诊断技术：包括尿液分析与尿液检查、影像学在泌尿外科的应用、泌尿系统内镜检查和泌尿
系统功能性检查等。

4. 泌尿外科手术技术：包括内腔镜手术（经皮肾镜、经输尿管肾镜）、肾脏手术（肾切除、部分切除）、膀胱
手术（膀胱切除、膀胱重建）、前列腺手术（前列腺切除术、经尿道前列腺电切术）和泌尿系统镜下手术技术
等。

5. 术后患者管理与康复：包括手术并发症处理及康复策略的病例分享与讨论。
课程的教学环节设计包括讲座与学术报告，专家学者介绍泌尿外科领域的最新研究进展，分享实际临床经验。此
外，还有实地参观，参观泌尿外科手术室和膀胱镜室，了解最新的医疗设备与技术，促进理论与实践结合。同
时，还有临床案例分析、技能培训、小组讨论和病例演示等教学环节，以培养学生的临床思维、解决问题的能
力，提高实际操作技能，并促进团队合作与创新思维。

综上所述，《泌尿外科学》课程以综合性教学为特点，将解剖学、生理学、疾病学、手术技术和护理等内容有机
结合，形成完整的知识体系，以满足学生的全面学科需求。通过实践操作、模拟手术等环节的训练，学生将在泌
尿外科领域具备实际操作技能。此课程还鼓励学生进行病例讨论和病例分享，促进团队协作与创新思维。通过教
学环节的合理设计，学生将更好地理解泌尿外科的临床实践。

### 课程内容简介

逻辑学是以有效推理形式为主要对象的工具性学科。逻辑学从古希腊和我国先秦以来，有着悠久的历史，逻辑学的现代形式是数理逻辑。本课程兼顾现代数理逻辑和传统形式逻辑，以数理逻辑的思路为纲，吸取传统逻辑中对日常推理有用的部分。内容涉及逻辑学的基础知识，包括命题、词项、推理、演绎、归纳、论证等，并简要介绍逻辑学发展史和现代逻辑发展情况，同时结合实际分析同逻辑有关的语言问题、同逻辑学有关的哲学问题。作为非逻辑学专业的通识课，本课程宗旨不在于要求熟练掌握这些具体内容，而在于通过这些具体内容，使学生对逻辑学的思路、精神和方法有所了解。

不定期当堂限时小测验；期末开卷作业。

### 课程内容简介

本课程带领学生纵览一系列逻辑学经典结论，包括哥德尔（一阶逻辑）完全性和（一阶
算术）不完备性定理、一阶逻辑不可判定性的丘奇-图灵证明、以及塔斯基的（形式算
术中）真之不可定义性定理。在回顾一阶逻辑的语法和语义之后，课程将介绍亨金借助
语法模型和极大一致集的对一阶逻辑完全性的证明，讨论该结论及其证法的哲学和逻辑
学影响，并引入一点模型论的内容。进而，课程介绍理论的完备、不完备、可判定等概
念。在了解20世纪初哲学和数学发展的背景（包括希尔伯特计划）之后，课程先在非形
式的水平上介绍不完备性定理及相关结论，包括其证明思路，然后再逐渐补充重要的细
节。课程总体上强调哲学和数学意义上的重要概念和思路，同时也为进一步学习技术细
节铺平道路。

### 课程内容简介

我们将以当代知识论的讨论为基础以逻辑学理论为工具对知识及其相关概念进行分析，考察各种理论。课程将以讲授加学生报告的形式进行。讲授部分主要介绍逻辑学理论，如、关于知识与信念的逻辑、反事实条件句逻辑等等，学生报告部分主要以当代认识论相关论文为主要内容。课程后半段将以讲授如何对当代认识论中关于知识的理论进行逻辑分析为主要内容。
